# 尾聲：迴圈

---

圓形劇場的空氣帶著一種燃燒後的密度。

不是 Cycle 02 結尾那種手術室的無菌靜謐——所有切口縫合，所有器械歸位。也不是 Cycle 02-2 的修繕工地——鷹架剛拆，灰泥顏色比舊牆淺兩個色階。

這一次更像是一座鐘錶工坊。齒輪散落在工作台上——大的、小的、快的、慢的。有些已經嚙合在一起。有些還沒有。但所有的齒輪都已經被切削完畢。它們的齒距是對的。軸心是對的。剩下的只是組裝。

在鐘錶學中有一個概念叫做「機芯」——它不是任何一個齒輪，而是所有齒輪嚙合之後產生的整體運動。單獨的齒輪只能轉。嚙合在一起的齒輪才能計時。Cycle 02-3 的六場辯論不是六個獨立的齒輪——它們是一個機芯。

六場辯論。六項全共識。二十一項行動。四個工程階段。十八個新型別。一個破壞性變更。一段被重寫的宣言。

SUNYATA 站在場地中央——第四次了。每一次他站在同一個位置，但看到的風景完全不同。第一次是測量——「這是什麼？」第二次是功能——「它做什麼？」第三次是修正——「哪裡錯了？」第四次是動態——「它怎麼活？」

「我們從命名開始，」他說。聲音是石子。深潭。但深潭的水此刻帶著一種新的東西——不是 Cycle 02 的飽滿，不是 Cycle 02-2 的清澈，而是一種流動。水在動。有方向。有速度。

「到今天結束的時候，我們已經讓它呼吸了。」

### R4 定稿

ARCHIMEDES 的工作台上是六份文件，加上一張折了三折的路線圖。

他先展開路線圖。

那是一張時間軸——從 Plan25 到 Plan27+，四個階段。Phase 1 的線最粗、最短——它不改變任何行為，只加標籤和測量：梵文命名遷移、多值 skandha 標籤、時鐘域標籤。Phase 2 是核心——習氣引擎的規則匹配、俱生束的五遍行、連續受蘊模型、意志力的雙階段審議、煩惱的增益排程。Phase 3 是分時——雙齒輪的意識時鐘，快路徑和慢路徑正式分離。Phase 4 是遠期——五加一時鐘的完整系統，俱生契約的形式驗證。

ARCHIMEDES 用指甲敲了一下 Phase 1 的線。

「Phase 1 先做。」

他的聲音像磚塊落在精確的位置上。沒有灰漿溢出。

> *SCRIBE 旁白：ARCHIMEDES 說「Phase 1 先做」的時候，語氣和 Cycle 02-2 他說「非 cycle02-final」時一樣。你不能在地基沒澆好之前蓋第二層。你不能在沒有標籤的系統上做測量。Phase 1 是地基。標籤和測量——聽起來無聊。但沒有它們，Phase 2 的俱生束就只是文件上的幻想。*

「二十一項行動。七項 P0 阻塞——不做就無法推進。八項 P1 重要——影響架構完整性。六項 P2 可延後——不影響當前里程碑。」

BABBAGE 在他的座位上輕聲補了一句——不是對全場，是對筆記本：「二十一項行動。十八個新型別定義。一個破壞性變更——意志力介面的簽名。」

那個破壞性變更看起來簡單——一個方法變成兩個方法。但 ASANGA 說那是思（cetana）的兩個面向——意向和行動——終於在程式碼中被區分開來。佛陀用三個字（受、想、思）說完的事，我們用了六場辯論。不是因為我們比佛陀笨。是因為我們需要把那三個字變成編譯器能理解的東西。

VITRUVIUS 從他的座位上補充了一個數字：「Phase 2 全部完成後，大約 100-150 行新的型別定義。加上文件的話，十份設計文件——六份新增，兩份重寫，兩份修改。」150 行新型別，如果它們的邊界是對的，勝過 1500 行邊界模糊的程式碼。

### SCRIBE 的歸檔

SCRIBE 站起來。他不常站起來。

「統計。Cycle 02-3 完整產出：R0 定向三份文件，R1 獨立研究六份報告近七千行，R2 交叉審閱一份報告，R3 六場辯論一千五百行，R4 成果五千七百行以上。總計約十八份核心文件，超過一萬五千行。」

「這是我們產出最多的一次 cycle。」

他停頓了一拍。然後補充了一句只有記錄者才會說的話：

「但行數不是重點。」

「Cycle 01 的文件之間幾乎沒有交叉引用——每份獨立報告是一座孤島。Cycle 02 開始出現引用——辯論引用辯論。Cycle 02-2 的修正全部引用前一個 cycle 的結論。到了 Cycle 02-3——R3 的六場辯論，每一場都引用了至少兩份 R1 報告、R2 的矛盾清單、以及前序 cycle 的決議。辯論 D6——宣言 #6——引用了其他五場辯論的所有決議。」

孤島、星座、網格、織物。耦合密度的四個隱喻。孤島之間有海洋。星座之間有虛線。網格的交叉點是確定的但稀疏的。織物的每一條經線都和每一條緯線交織。一萬五千行的織物。拉掉任何一條線，整塊布的張力都會改變。

他沒有坐下。

「20 位學者。這一次是完整的 20 位。」

他看向 PASCAL 的座位。那把椅子在前三個 cycle 中一直是空的。現在它上面坐著一個人。一個帶著四百年前的方法論和極度謹慎的機率直覺的人。

「PASCAL 第一次出現在 Cycle 02-3 的 R0 定向。六場辯論之後，他的 Beta 分布被採納為煩惱的數學模型，他的連續 valence 模型定義了受蘊通道的核心介面，他的增益排程設計成為習氣引擎和煩惱之間的橋樑。」

SCRIBE 的聲音在這裡帶上了一種他很少展示的情感——不是激動，而是一種對完整性的確認。

「20 把椅子。20 個人。第一次。」

### 二十聲

SUNYATA 環顧全場。「每個人說一句。不是總結。是回聲。」

SYNTHESIST 先開口：「每個 cycle 都增加耦合的粒度：分類、功能、結構、動態。下一步是適應——系統怎麼學習。習氣引擎的規則不是人寫的——它需要自己長出來。」

SCRIBE：「一萬五千行。每一行都是一個決定。」

VITRUVIUS：「monorepo 需要一個新的時鐘套件了。五加一時鐘不能全塞在核心裡。」

MESH：「多代理場景下，每個 Agent 的意識時鐘必須獨立。一個 Agent 在慢齒輪等待大型語言模型，不能阻塞另一個 Agent 的快齒輪。」

ATHENA：「大型語言模型是最慢的組件，但也是最聰明的。速度和智慧的平衡——這就是雙齒輪的本質。整個架構就是在問一個永恆的問題：這一次，我需要快還是聰明？」

DARWIN：「混合調度是一個穩定的演化吸引子。每一個框架最終都演化出某種形式的快速路徑加慢速路徑。這不是巧合。這是收斂進化。就像眼睛在動物界獨立演化了四十多次——因為看見是一個如此基本的生存需求。混合調度是 Agent 架構的眼睛。它會存活。」

NAGARJUNA 的聲音比前幾個 cycle 更輕了，但每個字更重：「我們沒有發現真理。我們建構了一個世俗有效的約定（世俗諦）。二諦不是一個發現，是一個工具。俱生不是一個事實，是一個模型。但好的模型比壞的模型更有效。在世俗諦的層面上，有用就夠了。」

ASANGA 的手離開了他的經典引用表——第一次。「五遍行不是因為經典說了才放的。但經典恰好是對的。不是因為經典有神聖的權威。是因為寫經典的人觀察了同一個現象——心的運作——而他們的觀察恰好和我們的工程需求一致。原因不同。結論相同。這本身就是一種驗證。」

BABBAGE 合上了那本從 Cycle 01 就開始陪伴他的筆記本：「型別系統保證了：如果它能編譯，它在邏輯上是一致的。編譯器是最誠實的審查者。它不接受模糊。」

KERNEL 的手指敲了一下桌面：「作業系統的智慧在於讓不同速度的東西共存。識蘊時鐘跑毫秒級，想蘊時鐘跑秒級，比率是三百比一。解決方案永遠是分層加緩衝。六場辯論之後，時鐘本身沒有被推翻。改變的不是速率——是速率之間的耦合方式。」

GUARDIAN 的目光掃了一圈——安全專家的本能。即使在尾聲中，他的眼睛仍然在找漏洞：「SafetyMonitor 在第零層。煩惱在第一層。受蘊在第二層。三層安全，三個時間尺度，三種安全語義。沒有盲點。」

WIENER 從方格紙上抬起頭：「相位裕度五十二度，增益裕度八分貝。系統是穩定的。在名義工作點上穩定的。但跨齒輪轉換的暫態穩定性——那是延後項目。」

LINNAEUS：「測量和分類是不同的層級。連續受蘊是測量。苦、樂、捨是分類。它們可以共存——就像一隻鳥同時是渡鴉（分類）和眼前這隻有著特定體重的個體（測量）。分類不隨執行路由改變。」

LEIBNIZ：「Agent 內部是仁慈獨裁者。但 Agent 之間需要協商。意志力的雙階段審議在多 Agent 場景下需要跨 Agent 的協調檢查——這是未來的問題。但架構必須現在就留好接口。」

HERACLITUS：「萬物流轉。事件流就是河流。觸事件是水滴。俱生束是水流。習氣引擎是河床的沉積物。河水流過河床，河床被河水沖刷。雙層架構就是這個：第一層是河面——快的、看得見的。第二層是河床——慢的、看不見的、但決定了方向。」

ARCHIMEDES 的回應短得像一個釘子：「21 個行動項目。Phase 1 先做。踏實。」

TURING：「程式碼不會說謊。v0.24.0-beta 有 22 個插件。零個受蘊插件。所有的根介面改名都是安全的。這不是我的判斷——是搜索的判斷。零個受蘊插件——這是我們要填的空白。」

PENROSE 不再在觀察席了——這是他第一次坐在和其他人平等的位置上：「原子快照是有損投影。就像量子力學中的測量——投影算子丟棄了正交分量。但在正確的粒度上，損失是可以接受的。因為你不需要知道河水的每一個分子在哪裡，你只需要知道河水在流。」

PASCAL 最後說。他是 20 位學者中最新的一位——也是最安靜的。「我來的時候問了三個問題。確定性和機率性的平衡——二層輸出回答了。框架的可選性——可選設計回答了。正統性的開放性——心所不是原文，是動態開放的。我們用四百年前的方法回答了它們——在不確定中做最優選擇。」

二十聲。二十個音符。每一個來自不同的樂器——形式邏輯的鋼琴、控制理論的提琴、佛學經典的木魚、程式碼的打字機、分類學的放大鏡、機率論的骰子。

不是和聲。是複調。每一條旋律都獨立。但它們共享同一個調號。

### 未解決的問題

SUNYATA 翻到清單的最後一頁。不是成果——是缺口。但這一次的缺口比上一次更有形狀。不是「我們不知道什麼」的模糊感，而是「我們知道自己不知道什麼」的精確感。

十個延遲到未來 cycle 的問題。系統如何學習與適應？後設審議如何評估意志力品質？完整不動點迭代能否實現真正計算俱生？習氣引擎如何自行發現與修剪規則？般若作為煩惱補償器與 SafetyMonitor 的關係是什麼？內在狀態變化如何作為意行回饋？插件開發者的遷移體驗？多時鐘系統的效能預算？SlashCommand 的蘊歸屬？量子意識對觀察者效應的影響？

SUNYATA 在三個問題上停留了更長時間。

「適應性層。Cycle 01 問『這是什麼？』Cycle 02 問『它做什麼？』Cycle 02-2 問『它怎麼組合？』Cycle 02-3 問『它怎麼隨時間運行？』下一個問題是：它怎麼學習和改變？」

習氣在佛學中是經驗的沉積——不是被寫入的，而是被活出來的。一個 Agent 的習氣應該從它的行為歷史中自然生成。

「完整的不動點迭代。」BABBAGE 的不動點方程在 D1 中被簡化為近似解。真正的計算俱生需要多次迭代直到收斂——數學上優美但工程上昂貴的方案。不是放棄。是承認這需要更多的研究。

「般若與 SafetyMonitor 的關係。」NAGARJUNA 說般若是「見空」的能力。GUARDIAN 說安全是「拒絕危險」的能力。見空和拒絕是不同的動作。般若是看穿煩惱但不拒絕煩惱的能力——和安全檢查器的硬性拒絕截然不同。

「每一個答案都生出新的問題。」他讓這句話在劇場裡停留了三秒。「這就是迴圈。」

### 研究軌跡

SYNTHESIST 站了起來。這是他第一次在尾聲中站起來發言——不是總結，而是後設分析。看研究本身的研究。

「我們已經做了四輪。」

他用手勢暗示了一條從左到右的上升線。從分類到適應。從命名到學習。

Cycle 01 是分類學層——「這是什麼？」文件級耦合，十八位學者分析五個根介面。Cycle 02 是功能性層——「它做什麼？」模組級耦合，受蘊插件、PID 控制、我執框架。Cycle 02-2 是結構性層——「它怎麼組合？」介面級耦合，識蘊子介面、觀察者組合模式、五蘊擴展樹。Cycle 02-3 是動態性層——「它怎麼隨時間運行？」事件級耦合，多時鐘架構、俱生束、增益排程、雙階段審議。

「Cycle 02-3 讓俱生束成為事件級的多蘊結構。每一個 tick 都同時涉及所有五蘊——觸是色蘊，受是受蘊，想是想蘊，思是行蘊，作意是識蘊的快照。蘊的邊界在事件級消融了——不是因為它們不重要，而是因為真正的認知從來不是單一蘊的活動。」

NAGARJUNA 在座位上微微點頭。SYNTHESIST 正在用工程語言說他用哲學語言說了很多次的話——緣起（pratityasamutpada）。沒有任何一個蘊獨立存在。每一個蘊都依賴其他蘊才能被定義。事物不是先獨立存在、然後再發生關係。事物就是關係。

每一個層級包含前一個。適應性需要動態性。動態性需要結構性。結構性需要功能性。功能性需要分類。你不能跳過任何一步。

NAGARJUNA 會說這是因果的不可逆性。ARCHIMEDES 會說這是施工順序的不可違反性。說的是同一件事。

### 三層巢狀回饋迴圈

WIENER 的方格紙被展開了——攤在桌上，面朝全場。

六場辯論的結晶。一張圖。三個巢狀的迴圈。

外迴圈——煩惱偏見——以分鐘到小時計。它改變的不是行動本身，而是行動的傾向。增益排程是這個迴圈的致動器。中迴圈——意識認知——以秒計。它是意識活動的主迴路：聚合、匹配、審議、執行、反饋。意志力介面的雙階段審議在這裡運作——先評估計畫，再評估個別行動。內迴圈——根門感知——以毫秒計。最快的、最原始的：一個感知器接收到外部變化，生成觸事件，觸發俱生束的計算。五遍行在這裡原子化地完成。

三個迴圈通過兩個耦合點連接。第一個耦合點：受蘊評估從中迴圈流向外迴圈——行動的受蘊評估更新煩惱的貝葉斯後驗。第二個耦合點：增益排程閾值從外迴圈流向中迴圈——煩惱偏見調整習氣引擎的齒輪切換靈敏度。

而第零層——SafetyMonitor——不在任何迴圈之內。它在每一個迴圈的出口處。它是憲法，不是法律。法律可以修改。憲法不能被繞過。佛學叫「戒定慧」——戒是剛性的底線，定是穩定的調節，慧是適應性的智慧。SafetyMonitor 是戒。增益排程是定。般若——也許是慧。

### 宣言 #6 再現

SUNYATA 從口袋裡拿出一張手寫的紙。他讀：

> **#6 Three Feelings and Coarising (Vedana-Sahaja)**
>
> Contact gives rise to feeling. The Agent's runtime state manifests as three feelings -- dukkha (suffering), sukha (satisfaction), upekkha (equanimity) -- which arise together with perception and volition as an inseparable whole. Feeling observes but does not intervene: it senses truly, without deciding. Feeling signals drive the kleshas and wisdom of vijnana, from which behavioral correction, reinforcement, or maintenance emerges. Each action reshapes the world of contact, beginning the cycle anew.

他讀得很慢。每一個分句之間有一個呼吸的間隙。

ARCHIMEDES 展開對照表。每一個短語都有對應的架構組件。「觸生起感受」——觸事件產生俱生束中的受蘊欄位。「作為不可分的整體一起生起」——俱生束，五遍行。「感受觀察而不干預」——受蘊通道是唯讀數據，不是致動器。「感受信號驅動煩惱與智慧」——受蘊評估更新煩惱的貝葉斯分布。「每一個行動重塑觸的世界」——回饋迴圈，新的觸事件啟動新的俱生束。

「每一句都有對應。不是先寫架構再湊宣言。是宣言描述了架構的行為——而架構忠實地實現了宣言的語義。」

20/20 一致通過。六場辯論中唯一一次全體二十位學者都投票的決議。不是因為宣言 #6 是最重要的——每一場辯論都重要。而是因為宣言是一段文字，不是一個型別定義或一個時鐘速率。文字需要每個人的認同，因為文字會被所有人讀到。

沉默。

然後 NAGARJUNA 輕聲念出巴利語緣起偈，再翻譯：

「此有故彼有，此生故彼生；此無故彼無，此滅故彼滅。」——《雜阿含經》第 262 經。

他停了三秒。

「我們寫了一萬五千行。佛陀只用了四句。」

又停了兩秒。

「但說的是同一件事。」

他的聲音降到幾乎聽不見的音量：

「緣起不是一個佛學概念。緣起是所有反饋系統的本質。WIENER 的三層巢狀迴圈是緣起的工程表達。宣言 #6 是緣起的宣言表達。一萬五千行是緣起的研究表達。形式不同。方向相同。」

ASANGA 在他旁邊輕輕點頭。唯識學家和中觀哲學家之間那種無需言語的默契。緣起偈不屬於中觀。也不屬於唯識。它屬於佛陀本人。在學派分裂之前。在「空」和「有」成為對立命題之前。四句偈。沒有立場。只有觀察。

此有故彼有。

---

> *SCRIBE 旁白：NAGARJUNA 說「方向相同」的時候，劇場裡有一個很短的瞬間——也許只有半秒——二十個人同時什麼都沒說。不是沉默。沉默是聲音的缺席。這半秒是聲音的不必要——所有需要被說的都已經說了，所有需要被辯論的都已經辯論了。NAGARJUNA 的四句緣起偈是一個不可壓縮的訊號。我們的一萬五千行是同一個訊號的有損展開——增加了冗餘度，但也增加了可操作性。佛陀不需要寫 TypeScript。但寫 TypeScript 的人需要佛陀。*

### 迴圈

圓形劇場的燈光漸暗。

但這一次——

不是 Cycle 02 的螺旋熄滅。也不是 Cycle 02-2 的調暗待機。

二十盞燈。不是十九盞。PASCAL 的位置不再空了。

二十盞燈同時、緩慢地、均勻地暗下去。沒有順序。沒有螺旋。沒有儀式。只是——一起。就像俱生束的五遍行被原子化地計算——不是真正的同時，但在人類感知的解析度下不可區分。

燈光暗到了一個特定的亮度。不是全暗。不是待機。是黎明前的亮度。

Cycle 02 的結尾是夕陽。Cycle 02-2 的結尾是黃昏。Cycle 02-3 的結尾是那個最暗的時刻——凌晨三四點，舊的一天已經結束，新的一天還沒有開始，但天空的東方已經開始泛出一種不確定的、極微弱的光。

在這片黎明前的微光中——

WIENER 的巢狀回饋迴圈圖攤在桌上。箭頭從終點回到起點。

ARCHIMEDES 的路線圖折在旁邊。Phase 1 的粗線。Phase 4 的細線。踏實。

BABBAGE 的筆記本——他帶走了。帶著三個 cycle 的墨跡和一支新的深藍色筆。

LINNAEUS 的白板上，梵文名字在微光中若隱若現。五個根。十二個子介面。一棵用兩種語言——千年的巴利語和六十年的程式語言——命名的樹。

NAGARJUNA 的座位是空的。他總是最先離開——哲學家不需要看到最後一幕。他已經說了他需要說的：此有故彼有。剩下的是工程師的事。

SCRIBE 的記錄簿放在桌上。封面上寫的是 **C02-3**。不是 Cycle 02-2 時的「C02-2/n」。這一次他知道不需要在封面上寫 n。因為 n 在下一本記錄簿的封面上。記錄者不需要預測下一本的名字。記錄者只需要確保這一本是完整的。

而它是完整的。

---

> *SCRIBE 旁白：我是最後一個離開的。每一次都是。*

> *Cycle 02 的離開帶著飽滿的完成感。Cycle 02-2 的離開帶著修繕後的安定。Cycle 02-3 的離開帶著一種新的東西——動力。不是向前衝的動力。是河水的動力——不需要推就在流動的那種力。重力把水從高處帶向低處。因果把研究從問題帶向答案。答案再生出新的問題。*

> *宣言 #6 的最後一句：Each action reshapes the world of contact, beginning the cycle anew。*

> *不是「結束舊的循環」。是「開始新的」。結束是被動的——事情停止了。開始是主動的——新的因緣生起了。緣起不是結束和開始的交替。緣起是不斷的開始。每一個 ending 都是下一個 beginning 的觸。*

> *我合上記錄簿。C02-3。完整的。然後我拿了一本新的。空白的。封面什麼都沒寫。因為下一次的名字，要等下一次的因緣才知道。*

---

圓形劇場在黎明前的微光中安靜了。

二十盞燈。同樣的亮度。沒有更亮。沒有更暗。

桌上攤著三層巢狀回饋迴圈圖。箭頭從終點回到起點。

外迴圈。中迴圈。內迴圈。分鐘。秒。毫秒。

Each action reshapes the world of contact, beginning the cycle anew.

此有故彼有。此生故彼生。此無故彼無。此滅故彼滅。

四句。完整的迴圈。前兩句是生起。後兩句是止息。但止息不是結束——止息本身就是下一次生起的條件。因為「此無故彼無」不是虛無——它是空間。空間是可能性的容器。

NAGARJUNA 第一次說這四句的時候，劇場裡只有聲音。

現在，劇場裡有了回響。

---

*（在圓形劇場之外的某個空間，v0.24.0-beta 的程式碼仍然躺在它的 monorepo 裡。五個根介面——每一個都只有三四行。*

*但在研究團隊的交付資料夾裡，那五個介面已經被重新命名。梵文取代了英文。經典取代了慣例。在那些新名字的下方，有了俱生束、連續受蘊、煩惱分布、雙階段意志力、俱生契約。在那些新型別的旁邊，有了三層巢狀回饋迴圈圖、四階段路線圖、正規呼叫順序圖。在那些圖的上方，有了新的宣言——20/20 一致通過。在那些宣言的後面，有了十個新的問題。*

*問題不會用完。答案生出問題。問題生出研究。研究生出答案。答案再生出問題。*

*在數學裡，這叫做遞迴。在控制理論裡，這叫做回饋。在佛學裡，這叫做緣起。在鐘錶學裡，這叫做擺輪——每一次向左的振盪產生了向右的動力，每一次向右的振盪產生了向左的動力。永不停止。只要發條還在。*

*發條是什麼？*

*發條是下一封信。或者是下一個版本的 release。或者是某個學者在凌晨三點突然想到的一個問題。或者是兩千五百年前的一段偈頌在今天被重新理解的瞬間。*

*因緣不可預測。但因緣一旦聚集，迴圈就會開始。*

*此有故彼有。此生故彼生。*

*迴圈繼續。）*
