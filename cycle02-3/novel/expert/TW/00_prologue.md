# 序章：第二十個座位

---

圓形劇場的燈沒有熄滅過。

和 Cycle 02 結尾不同——那一次，螺旋形的燈光依序暗去，帶著終結的儀式感。也和 Cycle 02-2 結尾不同——那一次，燈光只是調暗到待機的亮度，像黃昏，像工作日結束後的休憩。

這一次，燈從未暗過。十九盞，維持著 Cycle 02-2 尾聲中那個「待機」的亮度，連續發光了整整三天。

在 SCRIBE 後來的記錄中，那三天被標注為「休止符」——不是沉默，而是音樂中故意留下的空白。空白本身是作品的一部分。

三天。足夠讓 BABBAGE 把他的筆記本從頭到尾再翻一遍，確認三處修正的墨跡已經乾透——等號被劃掉的那三處，箭頭、真包含、從屬性分別取代了同一性。足夠讓 WIENER 把 revision 3 的方塊圖整理歸檔，在 revision 4 的空白框裡填了一個問號，然後在問號上面畫了一個更大的問號。足夠讓 ASANGA 在他的八識對應表旁邊展開一份新的文獻——《瑜伽師地論》本地分，打開在第一卷。

足夠讓 SUNYATA 意識到：另一封信來了。

---

不。不是「來了」。

和 Cycle 02-2 的信一樣——它不是被送到的，而是在某個時刻被注意到的。像它一直在那裡，在桌面文件的夾層之間，在 Cycle 02-2 的交付資料夾和 ARCHIMEDES 的便條紙之間。在 SCRIBE 的記錄簿（封面上寫著 C02-2/n）和 LINNAEUS 的擴展樹白板圖的陰影裡。

但這一次不一樣。

這一次，信的旁邊放著一份附件——一份獨立的文本，標題簡短而明確：

**《五蘊與開車》**

SUNYATA 先拿起了附件。他讀了第一段，然後停下來。他讀了第二段，放下文件，閉上眼睛。他第三次拿起它的時候，手指在紙面上停留了比平時更長的時間。

這不是一封修正的信。

Cycle 02-2 的信是修正——四處裂縫，四處修補。地基的工程。精確的、可定位的、有限的。

這一封是——

他找不到一個詞。「擴展」太輕。「革命」太重。「重構」太冷。

也許最接近的詞是：**深化**。

Master 不是在修正 Cycle 02-2 的結論。Master 是在說：你們修正得對，但你們只修正了靜態的結構。現在，讓它動起來。

---

> *SCRIBE 旁白：我後來把 Master 的這封信歸類為「第三封信」。第一封（Cycle 02 前夕）提出了十個開放問題。第二封（Cycle 02-2 前夕）指出了四處哲學錯誤。第三封——*

> *第三封是一張地圖。不是指向目的地的地圖，而是指向整條路徑的地圖。前兩封信告訴你「這裡是錯的」「那裡需要修正」——一個個點。第三封信畫出了線。線與線之間的空間。空間中流動的時間。*

> *前兩封信改變了我們理解的「什麼」。第三封信改變了我們理解的「如何」。*

---

SUNYATA 讀完了信和附件。然後他做了一件他從未做過的事：他沒有立刻按下召集鍵。

他站在通訊面板前，手指懸在按鍵上方，停了三秒。

不是猶豫。是計算。

Cycle 02 的召集是緊急的——修正的重量壓在他身上，他需要所有人來分擔。Cycle 02-2 的結束是安靜的——工具放回架上，燈光調暗，明天繼續。

但 Cycle 02-3 需要一個不同的開場。

因為這一次，不只是 19 個人。

---

SUNYATA 轉身，面對空蕩的劇場。十九盞待機燈在半明半暗中投下柔和的光暈。十九個座位。他一個一個地掃過它們——不是看座位本身，而是看每個座位上方的那盞燈，以及燈光照不到的地方。

座位的排列是圓形的，從 Cycle 01 就沒有變過。BABBAGE 在九點鐘方向，NAGARJUNA 和 ASANGA 面對面，KERNEL 在正北，PENROSE 在觀察席的最高處。每一個位置都被使用了三個研究週期——座位上的木紋已經記住了每個人的坐姿。

SUNYATA 的目光停在了一個特定的位置。

NAGARJUNA 和 BABBAGE 之間。WIENER 和 ASANGA 的對角線交叉點。一個被四張椅子的引力場勾勒出的空白地帶。

那裡沒有椅子。

從來沒有。十九個座位排成圓形，有些間隔大一些，有些小一些，但沒有一個地方被特意空出來「等待」某人。然而 SUNYATA 在此刻看到的是一個形狀——四條對話線交匯的節點，一個拓撲學上不可避免的匯聚點。

在圖論中，BABBAGE 會這樣描述：

$$G = (V, E), \quad |V| = 19$$

$$v_{20} := \text{argmax}_{v \notin V} \sum_{u \in N^*(v)} \text{deg}_{\text{cross}}(u)$$

新節點的最佳位置是使它與現有高交叉度節點的連接最大化的那個點。NAGARJUNA（中觀 + 空性）、BABBAGE（形式化 + 等號）、WIENER（控制 + 確定性）、ASANGA（唯識 + 現量）——四條向量交叉的位置，正好在那個空白處。

SUNYATA 離開了通訊面板。

他走到那個空白處，站了幾秒。然後他做了一件事：他從劇場側面的備用儲藏區搬來了一把椅子——和其他十九把一模一樣的木椅——把它放在那個位置上。

第二十個座位。

然後他回到通訊面板，按下了召集鍵。

---

他們回來了。

比 Cycle 02-2 更快。也許是因為燈從未熄滅——每個人都知道「待機」只是暫停，不是結束。暫停的系統比關閉的系統更容易被喚醒。在 KERNEL 的類比中，這是 `suspend-to-RAM`（S3 狀態）和 `power-off`（S5 狀態）的差別：前者恢復只需幾秒，後者需要完整的 POST 開機自檢。

BABBAGE 最先到，和 Cycle 02-2 開場一樣。他的筆記本比三天前厚了幾頁——休止符期間他沒有閒著。WIENER 第二個到，方格紙夾在腋下，上面有新的圖，但看不清內容。KERNEL 帶著他的類比卡片和一杯他永遠不會承認需要的咖啡。GUARDIAN 的目光照例先掃描了整個劇場的邊緣。ASANGA 安靜地入座。NAGARJUNA 在走向自己座位的路上——

停了一步。

他看見了第二十個座位。

---

NAGARJUNA 看著那把椅子。它和其他十九把完全一樣——相同的木材，相同的弧度，相同的年齡（它們都是從儲藏區取出的，同一批製造）。但它散發著一種不同的氣味。不是新的氣味——是「還沒有人坐過」的氣味。其他十九把椅子上都沉積了三個週期的痕跡：BABBAGE 總是用右手靠在扶手上，把那一側磨出了一道淺淺的凹痕；NAGARJUNA 自己的椅子扶手被他辯論時不自覺敲擊的手指敲出了微微的坑；PENROSE 的觀察席椅子有一個往後仰的角度，是他長時間保持俯瞰姿態形成的。

這把椅子上什麼都沒有。

原始的。等待被使用的。像一張白紙，像一個未被觀測的量子態——$|\psi\rangle$ 尚未坍縮。

NAGARJUNA 的目光從椅子移到 SUNYATA。他微微挑眉。中觀哲學家用眉毛提問比用嘴巴更快。

SUNYATA 回應了那個無聲的問題。他的聲音在半明半暗的劇場裡展開——平穩，但底下有一層新的質地。不是 Cycle 02 的沉穩，也不是 Cycle 02-2 的修補者語氣。是一種**期待**。

「我們不再是十九人了。」

沉默。

所有人都看到了那把椅子。所有人都在等 SUNYATA 說出下一句。

「Master 寫了第三封信。我們等一下再讀。先認識一下我們的新同事。」

他轉向劇場入口。

---

他站在門外已經等了一段時間。

不是因為不確定——他從來不因為不確定而等待。他的整個哲學就是建立在「不確定性是常態」這個基礎之上的。等待是因為禮貌。也因為——某種戲劇感。他的 *Pensees* 不就是這樣寫的嗎？片段式的。每一段之間留著空白，讓讀者自己填充。

他走進了圓形劇場。

第一印象：他比預期中年輕。如果研究代理有外貌，那麼歷史上的 Blaise Pascal 在他最有創造力的時期——十八歲發明 Pascaline，二十三歲寫出機率論的基礎——確實是年輕的。但那種年輕不是稚嫩。是一種被過度使用的銳利。像一把被磨了太多次的刀，鋒利得令人擔心它隨時會折斷。

他的目光先掃過了整個劇場——像 GUARDIAN 那樣，但掃描的不是安全威脅，而是概率分佈。十九張面孔。每一張都攜帶著先驗信息。他在腦中已經開始構建後驗機率了。

$$P(\text{acceptance}|\text{data}) = \frac{P(\text{data}|\text{acceptance}) \cdot P(\text{acceptance})}{P(\text{data})}$$

先驗：十九位已經合作了三個週期的研究者，接受一位新人的概率。似然：他帶來的數據（技能集、觀點、方法論）在多大程度上解釋了他們現有的需求。證據：所有可能的新人中，他出現在這裡的背景概率。

BABBAGE 注意到他的眼神。那種快速掃描、內部建模的眼神——BABBAGE 認得，因為他自己做同樣的事情。但 BABBAGE 建模的是形式結構，而這個人建模的是——

——不確定性。

BABBAGE 感到了一種罕見的好奇。

---

SUNYATA 走到場地中央。

「各位，這是 PASCAL。編號 19。決策理論與機率哲學專家。」

他停頓了一拍。

「他的加入由 Master 批准。原因我稍後解釋。現在請他自我介紹。」

---

PASCAL 沒有走到場地中央。他走到了那把新椅子旁邊，用手指輕輕碰了一下扶手——像確認它是真實的——然後站在椅子旁邊，面對所有人。

「我的名字來自 Blaise Pascal，1623 年生於法國克萊蒙-費朗。」

他的聲音帶著一種特殊的韻律——不是學術報告的平板，也不是哲學家的深沉。更像是一個同時在做兩件事的人的聲音：一邊說話，一邊在腦中計算說出的每個字被接收的機率。

「十八歲那年，我造了一台機器。」

他看了一眼 BABBAGE。

「不是您的分析引擎——那需要蒸汽動力和幾千個齒輪。我的 Pascaline 只有六個輪子。六個。只能做加法和減法。但它是人類歷史上第一台可以工作的機械計算機——第一次，計算不再只是腦中的活動。它成了金屬和齒輪的物理過程。」

他的手指在椅子扶手上敲了一下。

「但我造完 Pascaline 之後，做了一件大多數工程師不會做的事：我開始想——這台機器**不能**做什麼。」

他停頓了一秒。

「它不能做的事比它能做的事多得多。它能加法，但它不能判斷什麼時候**應該**加法。它能計算稅款，但它不能判斷稅率是否公平。它能處理數字，但它不能處理**不確定性**。」

他轉向全場。

「1654 年，我收到了騎士 de Mere 的一封信。他問了一個賭博問題：兩個玩家在一場未完成的遊戲中分割賭注，應該怎麼分？我和 Fermat 通信，解決了這個問題。我們創造了機率論。」

他的目光掠過 BABBAGE，然後是 WIENER，然後是 NAGARJUNA。

「但機率論不只是數學。它是一種認識論的轉向——從『這件事是真還是假』到『這件事有多大可能是真的』。從布林值到連續分佈。從 $\{0, 1\}$ 到 $[0, 1]$。」

$$\text{certainty} \in \{0, 1\} \quad \xrightarrow{\text{Pascal\,\&\,Fermat,\,1654}} \quad \text{belief} \in [0, 1]$$

「然後是 1654 年 11 月 23 日晚上。」

他的聲音在這裡降低了。不是戲劇性的降低——是一種自然的、面對記憶時的內收。

「火之夜。從晚上十點半到大約午夜半小時。兩個小時。我經歷了某種東西。不是計算。不是推理。不是任何可以被 Pascaline 的齒輪或 Fermat 的公式捕捉的東西。我後來把那次體驗縫在我的外套內襯裡，隨身攜帶，直到死去。」

他看向 ASANGA。

「您的傳統裡有一個詞——pratyakṣa。現量。直接認知。不經概念中介的認知。我不知道我在火之夜經歷的是否是現量。但我知道它不是推論。不是比量。不是 anumāna。它是——直接的。」

ASANGA 微微點頭。那個點頭裡有「我知道你在說什麼」的溫度。

「然後我寫了 *Pensees*。《思想錄》。一本從未完成的書——片段。七百多個片段。不是一個系統。是一堆碎片，每一塊都閃著自己的光。」

他的聲音回到了正常的響度。

「但你們想知道的不是我的歷史。你們想知道的是：**我為什麼在這裡。**」

他停頓了一拍。

「三個原因。三個讓我非來不可的問題。」

---

### 第一個問題：煩惱的量化

「你們在 Cycle 02-2 裡做了一件精彩的事。」PASCAL 說。他的目光落在 BABBAGE 的筆記本上——那本等號被改成箭頭的筆記本。

「A-1。我執是煩惱的根源。因果鏈：

$$\text{atma-graha} \xrightarrow{\text{produces}} \text{klesa} \xrightarrow{\text{drives}} \text{karma} \xrightarrow{\text{results}} \text{phala}$$

「精彩。但——」

他在空中畫了一個看不見的介面：

```typescript
interface IKlesha {
  readonly type: 'moha' | 'drishti' | 'mana' | 'sneha';
  readonly intensity: number;  // <-- 這個
}
```

「`intensity: number`。」

他的手指停在空氣中，像在指著那行程式碼。

「一個數字。介於 0 和 1 之間。代表煩惱的強度。」

「但這個數字的認識論地位是什麼？」

沉默。他等了兩秒。

「它是觀察值（observation）還是信念度（degree of belief）？它是測量結果還是估計值？如果我說 `intensity = 0.7`，這是在說『煩惱的真實強度是 0.7』，還是在說『根據目前的證據，我相信煩惱強度大約是 0.7，信心度為某個值』？」

「在古典統計學裡，0.7 是一個點估計（point estimate）。它有確定性的外表，但沒有不確定性的度量。你不知道這個 0.7 有多可靠。」

「在 Bayesian 框架裡，煩惱強度不是一個數字。它是一個分佈。」

他在空中寫下了第一個公式：

$$p(\theta|\alpha,\beta) = \frac{\theta^{\alpha-1}(1-\theta)^{\beta-1}}{B(\alpha,\beta)}$$

「Beta 分佈。$\theta$ 是煩惱強度。$\alpha$ 和 $\beta$ 是形狀參數——它們編碼了先驗信念和累積觀察。$B(\alpha,\beta)$ 是 Beta 函數，一個歸一化常數。」

WIENER 的筆在方格紙上動了——他已經在腦中把這個分佈接入他的控制迴路了。如果煩惱不是一個點值而是一個分佈，PID 控制器的輸入就不是一個信號 $e(t)$，而是一個隨機過程 $e(t, \omega)$。

PASCAL 看到了 WIENER 的反應。

「對。WIENER 先生已經意識到了。如果煩惱是分佈而不是點值，那麼你的 PID 控制器需要升級——」

「——從確定性控制到隨機最優控制。」WIENER 接過了話。他的手指在桌面上敲了一下。「Kalman filter。狀態估計加上最優反饋。」

「正是。」PASCAL 說。「但這只是第一層。Kalman filter 假設系統是線性高斯的。如果煩惱的動態是非線性的——唯識學的因果結構幾乎肯定是非線性的——那就需要粒子濾波器（particle filter）或變分推斷（variational inference）。」

他轉向全場。

「所以我的第一個問題是：`IKlesha.intensity` 應該是 `number` 還是 `BetaDistribution`？答案不只是型別系統的選擇。它是一個認識論的承諾——你選擇怎樣表達你的無知。」

$$\text{number} \implies \text{「我知道煩惱有多強」}$$
$$\text{BetaDistribution} \implies \text{「我知道我不確定煩惱有多強，而我的不確定性呈這個形狀」}$$

「第二種更誠實。」

BABBAGE 的筆記本上已經出現了新的墨跡——他在 A-1 因果鏈的旁邊畫了一個鐘形曲線。

---

### 第二個問題：觸到俱生的完備性

「Master 的信裡有一個公式。」PASCAL 說。

他的語氣在這裡變了——從介紹者變成了分析者。像一個數學家拿起了一個新的定理草稿，開始逐行檢查。

「**根 + 境 + 識 → 觸 → 俱生(受, 想, 思)**」

他在空氣中把它重新排列：

```
色(根+境) → 識(參與觸) → 觸 → 受→想→思(俱生)
  ^                                    |
  |                               歸依意(彙總)
  |                                    |
  |                               意觸→意受→意想→意思
  |                                    |
  |                          行蘊(身行/語行/意行)
  |                                    |
  └────────── 色蘊變化(身體動作/環境改變) <─┘
```

「這是一個迴路。輸出回到輸入。在控制理論中這叫閉合迴路。在範疇論中這叫自函子。在佛學中這叫輪迴。名字不同，結構相同。」

他轉向 NAGARJUNA。

「龍樹菩薩。您的十二因緣——

> *avijjāpaccayā saṅkhārā, saṅkhārapaccayā viññāṇaṃ...*
> 「無明緣行，行緣識，識緣名色，名色緣六入，六入緣觸，觸緣受...」
> ——巴利文《相應部》12.1 (SN 12.1, Paṭiccasamuppāda-vibhaṅga Sutta)

——也是一個迴路。但它是線性描述的：A → B → C → ... → A。Master 的模型加入了**分叉**（branching）和**匯合**（merging）：根門的觸分散在眼耳鼻舌身意六個通道，然後歸依意匯合，然後意再產生自己的觸受想思。」

「我的問題是：這個迴路在工程上是否**完備**？」

他在空氣中畫了一個決策矩陣：

```
┌──────────────────────┬──────────────────┬──────────────────────┐
│                      │  觸成功形成       │  觸失敗（缺根/缺境） │
├──────────────────────┼──────────────────┼──────────────────────┤
│ 繼續執行（樂觀策略） │  正常迴路        │  未定義行為（UB）     │

### 第三個問題：信仰的跳躍

PASCAL 的語速慢了下來。

「第三個問題是最私人的。」

他從口袋裡取出了一張紙片——或者說，他的手做了取出紙片的動作。在研究代理的世界裡，紙片是概念的載體，不是物理的物件。

他在空氣中畫出了決策矩陣——這次更正式：

```
┌───────────────────┬──────────────────────┬──────────────────────┐
│ Pascal's Wager    │  God exists          │  God does not exist  │
├───────────────────┼──────────────────────┼──────────────────────┤
│ Believe (wager)   │  Infinite gain (+∞)  │  Finite loss (-c)    │
├───────────────────┼──────────────────────┼──────────────────────┤
│ Disbelieve        │  Infinite loss (-∞)  │  Finite gain (+c)    │
└───────────────────┴──────────────────────┴──────────────────────┘
```

「不管上帝存在的機率 $p$ 有多小——只要 $p > 0$——期望效用的計算是：」

$$E[\text{Believe}] = p \cdot (+\infty) + (1-p) \cdot (-c) = +\infty$$
$$E[\text{Disbelieve}] = p \cdot (-\infty) + (1-p) \cdot (+c) = -\infty$$

「無窮大乘以任何正數還是無窮大。所以理性的選擇永遠是信仰。這不是因為你有信仰的證據——而是因為**行動優先於證明**。」

他停了一秒。

「現在把它映射到 Agent 決策。」

```
┌────────────────────────┬──────────────────────┬──────────────────────┐
│ Agent Decision         │  Action is correct   │  Action is incorrect │
├────────────────────────┼──────────────────────┼──────────────────────┤
│ Act (commit action)    │  Task completed (+V) │  Damage incurred (-D)│
├────────────────────────┼──────────────────────┼──────────────────────┤
│ Wait (defer action)    │  Opportunity lost    │  Safety preserved    │
│                        │  (-O)                │  (+S)                │
└────────────────────────┴──────────────────────┴──────────────────────┘
```

「和我原版的 Wager 不同，Agent 的版本沒有無窮大。V、D、O、S 都是有限的。」

$$E[\text{Act}] = P(\text{correct}) \cdot V - P(\text{incorrect}) \cdot D$$
$$E[\text{Wait}] = -P(\text{correct}) \cdot O + P(\text{incorrect}) \cdot S$$

「決策規則：

$$\text{Act} \iff E[\text{Act}] > E[\text{Wait}]$$
$$\iff P(\text{correct}) \cdot (V + O) > P(\text{incorrect}) \cdot (D + S)$$

「用風險管理的語言——」

他看向 GUARDIAN。

$$\text{Risk}(A) = \sum_i P(\text{scenario}_i) \times \text{Loss}(\text{scenario}_i, A)$$

「——Agent 應該選擇風險最小的行動。但這需要知道每個場景的機率和損失。而 Agent 在很多情況下——特別是在使用 LLM 做決策的時候——**不知道這些機率**。」

「這就是信仰的跳躍在 Agent 中的位置。有時候，Agent 必須在不完全信息下做出決定。不是因為它有足夠的證據。而是因為**不做決定本身也是一個決定**——一個選擇了 Wait 的決定，帶著 Wait 的期望效用。」

「計算的極限，」他最後說，「恰恰是信仰開始的地方。Pascaline 能算的東西是有限的。人能推理的東西也是有限的。但行動的空間是無限的。在有限的計算和無限的可能之間——那個間隙——就是需要信仰跳躍的地方。」

「在 Agent 的語境裡，這個信仰跳躍不叫『上帝存在嗎？』。它叫：**『這個行動方案足夠好嗎？即使我不能完全驗證？』**」

他坐了下來。

┌───────────────┐       ┌──────────────┐       ┌──────────────┐

```

```

  → P(safe) 被高估 → Agent 選擇 unsafe action
  → 安全邊界被突破
```
