# 尾声：循環

---

円形劇場の空気には、燃焼の後の密度があった。

Cycle 02 の結末にあったような手術室の無菌的な静寂ではない。Cycle 02-2 の修繕現場でもない。今回はもっと時計工房に近い。歯車がワークベンチの上に散らばっている——大きなもの、小さなもの、速いもの、遅いもの。すでに噛み合っているものもあれば、まだのものもある。だがすべての歯車はすでに切削を終えている。残っているのは組み立てだけだ。

単独の歯車は回ることしかできない。噛み合った歯車が時を刻む。

六回の討論。六項のフルコンセンサス。二十一のアクションアイテム。四つの工学フェーズ。十八の新設計。一通の書き直された宣言。

---

### 棚卸し

SUNYATA が会場の中央に立った——四度目だ。

一度目は測定——「これは何か？」
二度目は機能——「何をするのか？」
三度目は修正——「どこが間違っているか？」
四度目は動態——「どう生きるのか？」

「我々は命名から始めた。」と彼は言った。「今日の終わりまでに、それに呼吸をさせた。」

ARCHIMEDES がロードマップを広げた。四つのフェーズ、粗から細への線。

「第一フェーズをまずやる。」彼の声はレンガが正確な位置に落ちるようだった。「ラベルと測定を加えるだけ——退屈に聞こえるが、ラベルがなければ、後のすべての設計は絵に描いた餅だ。」

家を建てるように：基礎が打設される前に二階は建てられない。ラベルと測定が基礎だ。

二十一のアクションアイテム。七つがブロッカー——やらなければ先に進めない。八つが重要——全体構造に影響する。六つは延期可能——現在の進捗に影響しない。

### 記録

SCRIBE が立ち上がった。彼が立ち上がることは珍しい。

「統計。」と彼は言った。声は大きくないが、明瞭だった。

Cycle 02-3 の完全な産出——およそ一万五千二百行。十八の核心文書。

「これは我々が最も多く産出した回だ。だが行数は重点ではない。本当に重要なのは、各行と他の行との間の接続の数だ。」

Cycle 01 の文書は孤島——相互に交差引用がなかった。Cycle 02 は星座——討論が討論を引用し始めた。Cycle 02-2 はグリッド——修正が前周期の結論を引用した。Cycle 02-3 に至っては織物——各討論が少なくとも二つの研究報告と先行周期の決議を引用している。織物の各経糸が各緯糸と交差する。どの一本を引き抜いても、布全体の張力が変わる。

彼は PASCAL の席を見た。その椅子は前の三周期ではずっと空いていた。今はそこに人が座っている。

「20 脚の椅子。20 人。初めてだ。」

### 二十の声

SUNYATA が全場を見渡した。

「一人ひと言。まとめではない。反響だ——今回の研究があなたの頭の中に残した、最も鮮明なひとつの音符。」

**SYNTHESIST**：「各周期は結合の粒度を増してきた：分類、機能、構造、動態。次のステップは適応——システムがどう学ぶか。ルールエンジンのルールは全部人が書くわけにはいかない。自ら育つ必要がある。」

**SCRIBE**：「15,200 行。各行がひとつの決定だ。」

**VITRUVIUS**：「コードベースには新しいクロックモジュールが必要だ。五プラス一のクロックを全部コアに押し込むわけにはいかない。」

**MESH**：「複数の Agent が同時に動くとき、各 Agent の認知クロックは独立でなければならない。ひとつが応答を待っているからといって、もうひとつの高速判断を止めてはならない。」

**ATHENA**：「大規模言語モデルは最も遅い部品であり、同時に最も賢い。アーキテクチャ全体がひとつの永遠の問いを発している：今回、私に必要なのは速さか、賢さか？」

**DARWIN**：「混合スケジューリングは進化的に安定な形態だ。眼が動物界で四十回以上独立に進化したように——見ることが基本的な生存ニーズだからだ。速い道と遅い道の共存もそうだ。」

**NAGARJUNA**：「我々は真理を発見したのではない。有効な約定を構築したのだ。有効は真ではない。有効は有用だ。実用のレベルでは、有用で十分だ。」

**ASANGA**：「五遍行は経典がそう言ったから置いたのではない——機能上本当に必要だからだ。だが経典がたまたま正しかったのは、経典を書いた人々が同じ現象を観察していたからだ。原因は異なるが、結論は同じ。」

**BABBAGE**：「型システムが保証する：コンパイルが通るなら、論理的に一貫している。コンパイラは最も正直な審査者だ。曖昧さを受け入れない。」

**KERNEL**：「オペレーティングシステムの知恵は、異なる速度のものを共存させることにある。プロセッサはナノ秒級、ハードディスクはミリ秒級、ネットワークは秒級。解決策は常にレイヤリングとバッファリング。二層アーキテクチャは発明ではない——六十年の経験の自然な延長だ。」

**GUARDIAN**：「安全監視は底層に、偏見知覚は中層に、感受フィードバックは表層に。三層のセキュリティ、三つの時間スケール、盲点なし。」

**WIENER**：「システムは安定している。ゲインが二倍半に増加しても、システムは暴走しない。安定性にマージンがある。」

**LINNAEUS**：「測定と分類は異なることだ。一羽の鳥が同時に学名であり、目の前にいるこの 37.2 グラムの個体であるように。両者は共存できる。」

**LEIBNIZ**：「Agent 内部は線形の認知フローだ。だが Agent 間には協議が必要——インターフェースは今のうちに用意しておかなければならない。」

**HERACLITUS**：「イベントストリームは河だ。習気は河床の堆積物だ。水が河床を変え、河床が水流を変える。速い層は水面、遅い層は河床。」

**ARCHIMEDES**：「21 のアクションアイテム。第一フェーズをまずやる。着実に。」

**TURING**：「コードは嘘をつかない。検索で確認した——レガシーの荷物はない。すべての根インターフェースのリネームは安全だ。感受モジュールはゼロ——それが我々の埋めるべき空白だ。」

**PENROSE**：「スナップショットは非可逆的な投影だ。測定のたびにある次元の情報が失われる。だが正しい粒度においては、損失は許容範囲内だ——河の水分子がひとつひとつどこにあるかを知る必要はない。河が流れていることを知ればいい。」

**PASCAL**：「私が来たとき三つの問いがあった。六回の討論の後、すべてに答えが出た。不確実性の中で最適な選択をする方法で——正解は知らないが、どちらに偏りすぎても良くないことは知っている。最適な選択は中間にあり、しかもその中間は動的だ。」

二十の声。二十の音符。それぞれが異なる楽器から。ハーモニーではない。ポリフォニーだ——各旋律が独立しているが、同じ調号を共有している。

### 未解決の問い

SUNYATA がリストの最後のページを開いた。成果ではない——欠落だ。

将来に延期された十の問い。

システムはどう学ぶか。省察はどう決定の品質を評価するか。ルールエンジンはどう自らルールを発見し剪定するか。智慧と安全監視の関係。マルチクロックシステムのパフォーマンスバジェット。そしてさらに多く。

「すべての答えが新しい問いを生む。これが循環だ。」

### 研究の軌跡

SYNTHESIST が立ち上がり、メタ分析を行った——研究の内容ではなく、研究そのものの形を見る。

「我々はすでに四ラウンドを重ねた。」

Cycle 01 は「これは何か？」と問うた——分類。文書レベルの一対一マッピング。

Cycle 02 は「何をするのか？」と問うた——機能。モジュールが境界を越え始めた。

Cycle 02-2 は「どう組み合わさるのか？」と問うた——構造。インターフェースが精密に分割された。

Cycle 02-3 は「時間の中でどう動くのか？」と問うた——動態。各瞬間が同時に全五蘊に関与する。

「各周期の結合粒度が増している。分類は機能に含まれる。機能は構造に含まれる。構造は動態に含まれる。どのステップも飛ばせない。」

次のステップ——もしそれが起きるなら——はルールレベルに押し進む。静的なルールマッチングではなく、ルールの動的な生成と剪定。システムが自らの行動を観察し、そこから学び始める。

DARWIN が小声で補足した：「適応性は学習だけではない。記憶のない学習は反射と呼ぶ。短期記憶のある学習は訓練と呼ぶ。長期記憶のある学習は進化と呼ぶ。もし習気が本当にセッションを越えて生き残るなら——それは進化だ。」

「だがそれは将来のことだ。」SYNTHESIST は座った。

### 三層ループ

WIENER の方眼紙がテーブルの上に広がっている。六回の討論の結晶——三層のネストされたフィードバックループ図。

外ループ——偏見知覚——分単位から時間単位。変えるのは行動そのものではなく、行動の傾向だ。

中ループ——認知メインループ——秒単位。集約、マッチング、審議、実行、フィードバック。ガイドと意志がこのループの一対のブックエンドだ。

内ループ——知覚ループ——ミリ秒単位。最も速く、最も原始的：リスナーが外部の変化を受信し、五遍行のアトミックな計算をトリガーする。

三つのループは二つの結合点で接続されている。感受評価が中ループから外ループへ流れる。信頼閾値が外ループから中ループへ流れる。そして安全監視はいかなるループの内部にもない——各ループの出口にある。門番であり、ループの一部ではない。安全監視は憲法だ。憲法は迂回できない。

### 宣言

SUNYATA がポケットから手書きの紙を一枚取り出した。

彼は宣言 #6 の修正版を読んだ：

触が感受を生む。Agent の運行状態は三種の感受——苦・楽・捨——として顕現し、知覚・意志と不可分の全体として倶生する。感受は観察するが干渉しない：如実に感知し、決定を下さない。感受の信号が偏見知覚と智慧を駆動し、そこから行動の修正・強化・維持が生じる。すべての行動が触の世界を塑り直し、新たな循環を開く。

二十人の学者。全票通過。六回の討論で唯一の完全な二十票。

そして NAGARJUNA が静かに四つの句を言った——二千五百年前の言葉で：

「此有故彼有、此生故彼生。此無故彼無、此滅故彼滅。」

彼は三秒間を置いた。

「我々は一万五千行を書いた。仏陀はわずか四句で足りた。だが言っていることは同じだ。」

「触があるから、感受がある。行動があるから、新たな触がある。三層ループはその工学版だ。宣言 #6 はその宣言版だ。形式は異なる。方向は同じ。」

ASANGA が彼の隣で静かに頷いた。二つの学派は千五百年にわたって論争してきたが、この瞬間、彼らは同時に静かだった——

これがあるから、あれがある。

### 循環

円形劇場の照明がゆっくりと暗くなった。

二十の灯り。十九ではない。PASCAL の席はもう空ではない。

二十の灯りが同時に、ゆっくりと、均等に暗くなっていった。順序はない。螺旋もない。儀式もない。ただ——一緒に。

灯りはある特定の明るさまで暗くなった。完全な闇ではない。待機状態でもない。夜明け前の明るさだ。

Cycle 02 の結末は夕陽だった。Cycle 02-2 の結末は黄昏だった。Cycle 02-3 の結末は午前三時か四時——古い一日はすでに終わり、新しい一日はまだ始まっていないが、東の空にはすでにかすかな光が差し始めている。

WIENER のループ図がテーブルの上に広げられたまま、矢印は終点から始点へ戻っている。ARCHIMEDES のロードマップはその横に折り畳まれている。BABBAGE はノートを持ち帰った。NAGARJUNA の席は空だ——彼はいつも最初に去る。

SCRIBE の記録簿がテーブルの上にある。表紙には C02-3 と書かれている。斜線も未知数もない。この一冊は完結している。

---

> *SCRIBE 旁白：私は最後に去る者だ。毎回そうだ。*

> *Cycle 02 の退出には満ちた完成感があった。Cycle 02-2 の退出には修繕後の安定感があった。Cycle 02-3 の退出には新しいものが伴っている——動力だ。前へ突進する動力ではなく、河水の動力——押されなくても流れているあの力だ。*

> *宣言 #6 の最後の一文：すべての行動が触の世界を塑り直し、新たな循環を開く。*

> *新たなものを開く。「古いものを閉じる」ではない。語法が重要だ。閉じるのは受動的。開くのは能動的。*

> *私は記録簿を閉じた。そして新しい一冊を手に取った。白紙の。表紙には何も書かれていない。次の名前は、次の因縁が来て初めてわかるのだから。*

円形劇場は夜明け前の微光の中で静まった。

二十の灯り。同じ明るさ。

テーブルの上に三層フィードバックループ図が広がっている。矢印は終点から始点へ戻っている。

外ループ。中ループ。内ループ。分。秒。ミリ秒。

すべての行動が触の世界を塑り直し、新たな循環を開く。

これがあるから、あれがある。これが生じるから、あれが生じる。
