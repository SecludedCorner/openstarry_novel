## 開車的完整循環

### 第一層：各根門的觸事件（局部循環）

你開車時，多個根門同時在跑：

```
眼根 + 前方紅燈 + 眼識 → 眼觸 → 俱生(眼受, 眼想, 眼思)
                                         │ 苦受(危險感)
                                         │ 眼想：取相「紅色圓形」
                                         │ 眼思：微細的趨避反應

耳根 + 喇叭聲 + 耳識 → 耳觸 → 俱生(耳受, 耳想, 耳思)
                                         │ 苦受(刺耳)
                                         │ 耳想：取相「後方車輛」
                                         │ 耳思：微細的警覺反應

身根 + 方向盤觸感 + 身識 → 身觸 → 俱生(身受, 身想, 身思)
                                         │ 捨受(平常)
                                         │ 身想：取相「握持狀態」
                                         │ 身思：維持握力
```

### 第二層：歸依意（彙總 + 意的二次加工）

```
         各根門的結果歸依意
                │
                ▼
意 + 法境(各根門的綜合結果) + 意識
                │
                ▼
           意觸 → 俱生(意受, 意想, 意思)
                    │ 意受：緊張感（綜合判斷後的苦受）
                    │ 意想：「紅燈 + 後車逼近 = 必須停車」
                    │ 意思：決定踩煞車（造作，這就是行蘊的核心）
```

### 第三層：思→身行（行蘊驅動色蘊）

```
意思(決定踩煞車)
        │
        ▼
   身行（行蘊）：命令腳踩煞車踏板
        │
        ▼
   色蘊變化：腳移動、踏板下壓、車輛減速
```

### 第四層：色蘊變化產生新的根境（下一輪循環開始）

```
車輛減速 → 產生新的境
        │
        ├─ 眼根：前方景物變化（車距縮短速度變慢）
        ├─ 身根：身體前傾的觸感（減速慣性）
        ├─ 耳根：引擎聲降低、煞車聲
        │
        ▼
   新的一輪觸 → 新的俱生(受, 想, 思) → 再歸依意 → ……
```

### 全貌：五蘊的完整迴路

```
色(根+境) → 識(參與觸) → 觸 → 受→想→思(俱生)
  ▲                                    │
  │                               歸依意(彙總)
  │                                    │
  │                               意觸→意受→意想→意思
  │                                    │
  │                          行蘊(身行/語行/意行)
  │                                    │
  └────────── 色蘊變化(身體動作/環境改變) ◄─┘
```

每一輪的「思」造作了行為，行為改變了色蘊（身體狀態、外在環境），改變後的色蘊又成為下一輪觸的新境。這就是五蘊不斷流轉的完整循環。

開車從出發到停車，就是這個迴路跑了無數輪。每一輪都是：**感知→判斷→行動→環境改變→新的感知**。這不是單次的 pipeline，而是一個**持續運轉的反饋迴路（Feedback Loop）**。

而修行要斷的，就是在「受→想」這個環節——如果你能在受生起的時候，不被想帶著跑進「紅燈→煩躁→罵人」的戲論鏈，迴路就不會往苦的方向轉。
