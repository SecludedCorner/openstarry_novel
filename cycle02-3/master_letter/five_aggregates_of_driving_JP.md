## 運転における五蘊の完全な循環

### 第一層：各根門における触の事象（局所的循環）

運転中、複数の根門が同時に作動している：

```
眼根 + 前方紅燈 + 眼識 → 眼觸 → 俱生(眼受, 眼想, 眼思)
                                         │ 苦受(危険感)
                                         │ 眼想：取相「赤色円形」
                                         │ 眼思：微細な趨避反応

耳根 + 喇叭聲 + 耳識 → 耳觸 → 俱生(耳受, 耳想, 耳思)
                                         │ 苦受(刺耳)
                                         │ 耳想：取相「後方車輛」
                                         │ 耳思：微細な警覚反応

身根 + 方向盤觸感 + 身識 → 身觸 → 俱生(身受, 身想, 身思)
                                         │ 捨受(平常)
                                         │ 身想：取相「把持状態」
                                         │ 身思：握力の維持
```

### 第二層：意への帰依（集約と意による二次加工）

```
         各根門の結果が意に帰依する
                │
                ▼
意 + 法境(各根門の綜合結果) + 意識
                │
                ▼
           意觸 → 俱生(意受, 意想, 意思)
                    │ 意受：緊張感（綜合判断後の苦受）
                    │ 意想：「赤信号 + 後続車の接近 = 停車の必要」
                    │ 意思：ブレーキを踏むと決定（造作、これが行蘊の核心である）
```

### 第三層：思から身行へ（行蘊が色蘊を駆動する）

```
意思(ブレーキを踏むと決定)
        │
        ▼
   身行（行蘊）：足にブレーキペダルを踏むよう命令する
        │
        ▼
   色蘊の変化：足の移動、ペダルの押下、車両の減速
```

### 第四層：色蘊の変化が新たな根境を生む（次の循環の開始）

```
車輛減速 → 産生新的境
        │
        ├─ 眼根：前方景物変化（車間距離の縮小速度が減少）
        ├─ 身根：身体が前傾する触感（減速慣性）
        ├─ 耳根：エンジン音の低下・ブレーキ音
        │
        ▼
   新たな一輪の觸 → 新たな俱生(受, 想, 思) → 再び意に帰依 → ……
```

### 全体像：五蘊の完全な回路

```
色(根+境) → 識(參與觸) → 觸 → 受→想→思(俱生)
  ▲                                    │
  │                               歸依意(彙總)
  │                                    │
  │                               意觸→意受→意想→意思
  │                                    │
  │                          行蘊(身行/語行/意行)
  │                                    │
  └────────── 色蘊變化(身體動作/環境改變) ◄─┘
```

毎回の「思」が行為を造作し、行為が色蘊（身体の状態・外的環境）を変化させ、変化した色蘊がまた次の触の新たな境となる。これこそが五蘊の絶え間ない流転の完全な循環である。

運転の出発から停車に至るまで、この回路は無数の循環を繰り返す。毎回の循環は：**感知→判断→行動→環境変化→新たな感知**である。これは単発の pipeline ではなく、**持続的に回転するフィードバック・ループ（Feedback Loop）**なのである。

そして修行において断ずべきは、まさに「受→想」という環節である——受が生起した刹那において、想に引きずられて「赤信号→苛立ち→罵倒」という戯論の連鎖に入り込まなければ、回路は苦の方向へ転じることがないのである。
