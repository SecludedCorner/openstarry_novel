## 开车的完整循环

### 第一层：各根门的触事件（局部循环）

你开车时，多个根门同时在跑：

```
眼根 + 前方红灯 + 眼识 → 眼触 → 俱生(眼受, 眼想, 眼思)
                                         │ 苦受(危险感)
                                         │ 眼想：取相「红色圆形」
                                         │ 眼思：微细的趋避反应

耳根 + 喇叭声 + 耳识 → 耳触 → 俱生(耳受, 耳想, 耳思)
                                         │ 苦受(刺耳)
                                         │ 耳想：取相「后方车辆」
                                         │ 耳思：微细的警觉反应

身根 + 方向盘触感 + 身识 → 身触 → 俱生(身受, 身想, 身思)
                                         │ 舍受(平常)
                                         │ 身想：取相「握持状态」
                                         │ 身思：维持握力
```

### 第二层：归依意（汇总 + 意的二次加工）

```
         各根门的结果归依意
                │
                ▼
意 + 法境(各根门的综合结果) + 意识
                │
                ▼
           意触 → 俱生(意受, 意想, 意思)
                    │ 意受：紧张感（综合判断后的苦受）
                    │ 意想：「红灯 + 后车逼近 = 必须停车」
                    │ 意思：决定踩刹车（造作，这就是行蕴的核心）
```

### 第三层：思→身行（行蕴驱动色蕴）

```
意思(决定踩刹车)
        │
        ▼
   身行（行蕴）：命令脚踩刹车踏板
        │
        ▼
   色蕴变化：脚移动、踏板下压、车辆减速
```

### 第四层：色蕴变化产生新的根境（下一轮循环开始）

```
车辆减速 → 产生新的境
        │
        ├─ 眼根：前方景物变化（车距缩短速度变慢）
        ├─ 身根：身体前倾的触感（减速惯性）
        ├─ 耳根：引擎声降低、刹车声
        │
        ▼
   新的一轮触 → 新的俱生(受, 想, 思) → 再归依意 → ……
```

### 全貌：五蕴的完整回路

```
色(根+境) → 识(参与触) → 触 → 受→想→思(俱生)
  ▲                                    │
  │                               归依意(汇总)
  │                                    │
  │                               意触→意受→意想→意思
  │                                    │
  │                          行蕴(身行/语行/意行)
  │                                    │
  └────────── 色蕴变化(身体动作/环境改变) ◄─┘
```

每一轮的「思」造作了行为，行为改变了色蕴（身体状态、外在环境），改变后的色蕴又成为下一轮触的新境。这就是五蕴不断流转的完整循环。

开车从出发到停车，就是这个回路跑了无数轮。每一轮都是：**感知→判断→行动→环境改变→新的感知**。这不是单次的 pipeline，而是一个**持续运转的反馈回路（Feedback Loop）**。

而修行要断的，就是在「受→想」这个环节——如果你能在受生起的时候，不被想带着跑进「红灯→烦躁→骂人」的戏论链，回路就不会往苦的方向转。
