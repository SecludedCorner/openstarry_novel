## The Complete Cycle of Driving

### Layer 1: Contact Events at Each Sense Gate (Local Cycles)

When you drive, multiple sense gates are running simultaneously:

```
Eye-faculty + Red light ahead + Eye-consciousness → Eye-contact → Co-arising(Eye-vedana, Eye-sanna, Eye-cetana)
                                                                     │ Dukkha-vedana (sense of danger)
                                                                     │ Eye-sanna: Grasps sign "red circle"
                                                                     │ Eye-cetana: Subtle avoidance reaction

Ear-faculty + Horn sound + Ear-consciousness → Ear-contact → Co-arising(Ear-vedana, Ear-sanna, Ear-cetana)
                                                                     │ Dukkha-vedana (harsh sound)
                                                                     │ Ear-sanna: Grasps sign "vehicle behind"
                                                                     │ Ear-cetana: Subtle alerting reaction

Body-faculty + Steering wheel touch + Body-consciousness → Body-contact → Co-arising(Body-vedana, Body-sanna, Body-cetana)
                                                                     │ Upekkha-vedana (neutral)
                                                                     │ Body-sanna: Grasps sign "gripping state"
                                                                     │ Body-cetana: Maintains grip force
```

### Layer 2: Converging on Mano (Aggregation + Secondary Processing by Mind)

```
         Results from each sense gate converge on mano
                │
                ▼
Mano + Dhamma-object (synthesized results from all gates) + Mano-consciousness
                │
                ▼
           Mano-contact → Co-arising(Mano-vedana, Mano-sanna, Mano-cetana)
                    │ Mano-vedana: Tension (dukkha-vedana after integrated assessment)
                    │ Mano-sanna: "Red light + car closing in behind = must stop"
                    │ Mano-cetana: Decides to brake (sankhara — this is the core of the Formations aggregate)
```

### Layer 3: Cetana → Bodily Action (Sankhara-khandha Drives Rupa-khandha)

```
Mano-cetana (decides to brake)
        │
        ▼
   Bodily action (sankhara-khandha): Commands foot to press brake pedal
        │
        ▼
   Rupa-khandha change: Foot moves, pedal depresses, vehicle decelerates
```

### Layer 4: Rupa-khandha Changes Produce New Sense Objects (Next Cycle Begins)

```
Vehicle decelerates → Produces new sense objects
        │
        ├─ Eye-faculty: Visual field changes (gap shortens more slowly)
        ├─ Body-faculty: Body tilts forward (deceleration inertia)
        ├─ Ear-faculty: Engine sound drops, braking sound
        │
        ▼
   New round of contact → New co-arising(vedana, sanna, cetana) → Converges on mano → ...
```

### The Full Picture: Complete Circuit of the Five Aggregates

```
色(根+境) → 識(參與觸) → 觸 → 受→想→思(俱生)
Rupa(faculty+object) → Vinnana(participates in contact) → Phassa → Vedana→Sanna→Cetana(co-arising)
  ▲                                    │
  │                               歸依意(彙總)
  │                         Converges on mano (aggregation)
  │                                    │
  │                               意觸→意受→意想→意思
  │                     Mano-contact→Mano-vedana→Mano-sanna→Mano-cetana
  │                                    │
  │                          行蘊(身行/語行/意行)
  │                    Sankhara-khandha (body/speech/mind)
  │                                    │
  └────────── 色蘊變化(身體動作/環境改變) ◄─┘
              Rupa-khandha change (bodily action / environment change)
```

In each round, cetana produces action; action changes rupa-khandha (bodily state, external environment); and the changed rupa-khandha becomes the new sense object for the next round of contact. This is the complete cycle of the Five Aggregates in continuous flux.

From the moment you start driving to the moment you park, this circuit runs countless rounds. Each round is: **perception → judgment → action → environment change → new perception**. This is not a one-shot pipeline, but a **continuously running Feedback Loop**.

What contemplative practice aims to sever is precisely the link between vedana and sanna -- if, at the moment vedana arises, you can avoid being carried away by sanna into the papanca chain of "red light → irritation → cursing," the circuit will not turn toward dukkha.
