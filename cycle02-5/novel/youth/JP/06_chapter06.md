## 10人、仏教学者ゼロ

D5 は特別な討論でした。

参加者はわずか10人。そして仏教学者はいませんでした。NAGARJUNA と ASANGA は自ら辞退したのです。NAGARJUNA の別れの言葉は興味深いものでした。「D4 は名前が定義に責任を持つべきだと証明した。D5 は工学設計が仏教の名前なしでも完遂できることを証明するだろう。この2つは同じくらい重要だ。」

D5 の議題は純粋な工学問題でした。D4 で改名されたインターフェースを、実際に動作する完全な仕様に仕上げること。師父の要求は「仕様を100%にして、工学チームに渡す」でした。

## コード考古学

TURING は D5 のために特別なレポートを用意しました。14のソースコードファイルに含まれるすべての設計パターンを分析したのです。各 timeout はいくつか？ 各 Registry はどう動くか？ 各種の失敗時にはどう処理するか？

彼はこのレポートを「コード考古学」と呼びました。

このレポートは討論の進め方を変えました。それまでの討論は「原則」の上に成り立っていました。原則には異なる解釈があり得ます。しかし TURING のレポートは「事実」の上に成り立っていたのです。事実は一つしかありません。

## 9つの決定

D5 は75分で9つの投票を行いました。テンポが速かったのは、各項目に事実的な根拠があったからです。

最も重要な決定をいくつか紹介します。

**IConfidenceAuditor は独立したスロットを使う。** モニターとは共用しない。理由：モニターは観察のみ（副作用なし）、監査器は AI モデルを呼び出す（副作用あり）。観察者と副作用を持つコンポーネントを混ぜるべきではありません。全会一致で可決。

**戻り値の型はデュアルモード。** 同期でも非同期でもよい。これは D5 で最も僅差の投票でした――5票対3票。議論の焦点は「デュアルモードが良いか悪いか」ではなく、「既存の設計パターンを破るべきか否か」でした。最終的に「一貫性」が勝ちました。工学においては、既存のコードとの一貫性を保つことが、完璧を追求するよりも重要なことが多いのです。

**timeout は200ミリ秒。** 監査が200ミリ秒以内に応答しなければ、スキップする（信頼度を修正しない）。既存のギア仲裁器と一致させました。全会一致で可決。

**監査器は1つだけ。** 複雑な複数監査器の集約戦略は採用しない。YAGNI 原則に従います――You Ain't Gonna Need It（使わないものは先に作るな）。6対2で可決。

**均等な重み。** 品質ベクトルの4つの次元――連続性、粒度、速度、均衡性――それぞれ0.25。制御理論の専門家 WIENER はこう言いました。「運用データがないとき、等しい重みが最も保守的な選択だ。」全会一致で可決。

## 最終仕様

9つの投票が終わった後、TURING はホワイトボードに最終的なインターフェースを書きました。

```typescript
export interface IConfidenceAuditor extends IVijnana {
  readonly id: string;
  audit(confidence: number, context: GearContext):
    ConfidenceAuditResult | Promise<ConfidenceAuditResult>;
}

export interface ConfidenceAuditResult {
  delta: number;       // [-0.05, +0.05] に制限される
  reasoning: string;   // 監査記録
}
```

「100%」と彼は言いました。

これはコンセプト設計ではありません。エディタを開いてすぐに書き始められる完全な仕様です。

## 仏教ゼロ

記録係の SCRIBE は、D5 の討論全体で仏教用語が使われた回数を集計しました。

ゼロ。

D5 は本プロジェクト史上初めて、仏教コンテンツがまったく含まれない討論でした。意図的に避けたのではなく、自然な結果です。timeout の値や失敗処理の戦略を議論しているとき、仏教の経典を引用する必要はないのです。

NAGARJUNA は討論終了後にやって来てこう言いました。「D4 は名前が定義に責任を持つべきだと証明した。D5 は――工学問題の中にはそもそも定義を必要としないものがあることを証明した。仕様さえあればよいのだ。」

これは矛盾していません。同じ天秤の表と裏です。一方は「あなたの名前は定義にふさわしいか？」と問い、もう一方は「あなたの仕様は十分に精密か？」と問います。D4 が前者を校正し、D5 が後者を完成させました。

両面とも必要です。どちらが欠けてもいけません。
