## II. 二つの疑わしい文書

Doc 16 と Doc 31 は異質だった。

## III. 三つの次元

ARCHIMEDES が標本を数えている間、NAGARJUNA、ASANGA、PASCAL は別の作業室で対応づけ境界原則を構築していた。三人。三つの学問。三つの次元。

PASCAL は頷いた。そして彼の次元を展開した。

---

PASCAL のホワイトボードは NAGARJUNA の隣にあった。彼は二次元マトリクスを描いた。

```
                 対応づけに価値あり    対応づけに価値なし
保持する         True Positive        False Include ← 累積的損害
除去する         False Exclude        True Negative
                 ↑ 一回限りのコスト
```

「標準的な混同行列だ。」彼は言った。「だが鍵は行列そのものではなく——損害の非対称性にある。」

彼は False Include の横に上向きの矢印を描いた。

「False Include——装飾的対応づけを残した場合。その損害は何か？ 将来のすべての読者が——数十人かもしれないし数百人かもしれない——この文書を読むたびに——一回かもしれないし十回かもしれない——各ページに現れる仏教ラベルのたびに——三つかもしれないし十かもしれない——そのたびに、読者は頭の中で二つの用語体系を維持しなければならない。śīla とは何か？ ああ、つまり Hard Rules のことか。ならなぜ最初から Hard Rules と呼ばないのか？」

彼は一つの式を書いた：

$$C_{FI} = N_{readers} \times R_{reads} \times P_{pages} \times \tau_{cognitive}$$

「累積的損害。読者数 × 読書回数 × ページ数 × 一回あたりの認知切り替えコスト。これは乗算だ——各項はすべて正で、積は増大するのみ。」

そして False Exclude の横にずっと小さな矢印を描いた。

「False Exclude——価値ある対応づけを除去した場合。損害は何か？ いつか、誰かが五蘊分類の仏教的背景を知る必要が出る。付録を検索する。見つかる。一回限りの検索コスト。累積しない。読者数で乗算されない。読書回数で乗算されない。」

$$C_{FE} = S_{search} \quad \text{（一回限り）}$$

「結論。」彼は大きな不等号を描いた。

$$E[C_{FI}] \gg E[C_{FE}]$$

「期待値分析：装飾的対応づけは負の期待値を持つ。疑わしい場合は REMOVE または RELOCATE に傾けるべきだ。」

---

ASANGA はずっと聴いていた。彼にはホワイトボードがない——思考の道具は言語であり、図形ではない。

「機能的判定。」彼は言った。声は唯識学特有の精密さを帯びていた——顕微鏡下で焦点を合わせるように。「唯識学には核心的な判定基準がある：因果効力（arthakriyā-sāmarthya）。ある概念は因果効力を持つか——つまりある結果を駆動したか？」

彼は立ち上がった。NAGARJUNA と PASCAL のホワイトボードの間に歩いた。

「仏教概念が五蘊ルートインターフェースの設計を駆動した——これには因果効力がある。五蘊分類 → PluginHooks の五つのカテゴリ。因果連鎖が明確だ。保持。」

「仏教ラベルが Hard Rules に śīla と貼られた——これには因果効力がない。Hard Rules は śīla と呼ばれたから硬性制約を持つのではない——その硬性制約は SafetyMonitor の実装に由来する。śīla というラベルはいかなる因果連鎖も変えない。除去。」

彼は空中に手で一本の線を引いた——因から果へ。

「判定基準は単純だ：**この仏教概念を除去したら、因果連鎖は断たれるか？** 是 → 保持。否 → 除去。」

PASCAL は顔を上げた。三つの次元の交点が見えた：

```
NAGARJUNA — 二諦（カテゴリー分析）
            世俗諦 vs 勝義諦 → 二つの言語を混同するな

PASCAL    — 損害非対称（決定理論）
            E[累積損害] >> E[検索コスト] → 疑わしい場合は除去

ASANGA    — 因果効力（唯識学）
            因果効力あり → 保持 / 因果効力なし → 除去
```

三つの学問。哲学、数学、仏教。三つの公理体系。三つの推論経路。

一つの結論。
