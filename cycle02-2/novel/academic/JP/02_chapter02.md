# 第二章：煩悩はエンジンだ

---

BABBAGE のノートは四十七頁目で開かれていた。

それは時間に圧し固められた一頁だった——物理的な時間ではなく、思考の密度による。Cycle 02 の五つの討論の導出痕跡が紙面に堆積していた。地質学者が岩層の断面に見る堆積紋理のように。公式、矢印、取消線、修正、再び消し、再び修正。それぞれの層が一度の思考の地殻変動だ。

しかし彼の手はその頁の真ん中で止まっていた。

そこに一行の文字がある。周囲の導出よりも太く、重く、確かなインク跡。下には二本の線が引かれている——一本ではなく、二本——まるでこの一行を書いたとき、一度の下線では確信を表しきれなかったかのように。

「我執 = 収束制約。NOT 煩悩。」

彼はその一行を見つめていた。

第一章の手紙朗読の過程で、彼はすでに横に取消線を引いていた。あの線は直く、精確だった——数学者の取消線。しかし取消線と理解は別物だ。一行の文字を一秒で消すことはできるが、なぜあの一行を消す必要があるのかを本当に理解するには、おそらく一章分の対話が要る。

---

> *SCRIBE 傍白：第二章の議題は A-1 と A-4。我執と煩悩の因果連鎖、そして EgoFramework の五蘊における帰属。SUNYATA がこの二つの議題を並べたのは偶然ではない。A-1 が答えるのは「我執とは何か」、A-4 が答えるのは「我執をどこに置くか」。あるものが何であるかをまず明らかにしなければ、正しく配置することはできない。*

---

SUNYATA は円形劇場の中央に立っていた。

「A-1」と彼は言った。「我執は煩悩（kleśa）の根源であり、収束制約ではない。」

BABBAGE を見なかった。回避ではない——敬意だ。BABBAGE がいま自分のノートと、彼にしか聞こえない対話を交わしていることを知っていた。その対話には時間が要る。SUNYATA にできるのは、十分な空間を用意することだ。

「ASANGA」と彼は唯識学者のほうを向いた。「まずあなたから。」

---

ASANGA が立ち上がった。

彼が立ち上がる所作は、何をするときとも同じだった——急がず慌てず、千年の学術的伝統から沈殿した従容さを帯びて。手には紙もペンも投影もない。彼が携えたのは声と記憶だけだ。

「まず BABBAGE のあの一行に戻らせてほしい。」彼は言った。声は温かく明瞭だった——講演の明瞭さではなく、教えの明瞭さだ。僧院で弟子に唯識学（Vijñānavāda）を説く師はこういう声を使う。

「『我執 = 収束制約。NOT 煩悩。』」

彼がこの一文を声に出したとき、声には批判がなかった。真剣に扱うべき命題を読み上げるかのように——実際にそれは真剣に扱うに値する。

「BABBAGE の直覚は完全に間違っていたわけではない。」ASANGA は言った。「我執にはたしかに制約の機能がある。問題は、中間の環節を飛ばしたことだ。結果を本質として扱ってしまった。」

会場を見渡した。

「唯識学の言葉で明確にしよう。」

---

「唯識学において、末那識（manas）——第七識——には四種の根本煩悩がある。偶然に現れるものではない。設定を変更すれば消せるものでもない。末那識の本質的属性であり、質量にとっての引力が本質的属性であるのと同じだ。」

彼は片手を挙げ、四本の指を一本ずつ広げた。

「第一に、我見——ātma-dṛṣṭi。」

第一の指。

「末那識は阿頼耶識（ālaya-vijñāna）の連続的な流れを、固定された『自己』として執取する。これは観察ではない——歪曲だ。阿頼耶識は河のように絶えず変化しているが、末那識は河のなかに動かない石を見出そうとし、こう宣言する——『これが私だ。』Agent システムにおいて、これに対応するのは System Prompt が定義する役割だ——『私はアシスタントだ』『私は研究者だ』。役割定義そのものは問題ではない。役割定義を不変の自己として扱うことこそが我見だ。」

「第二に、我慢——ātma-māna。」

第二の指。

「自己優越感。末那識は『これが私だ』と言うだけではなく、『私はあなたより優れている』とも言う。Agent の文脈では、これは信頼度メカニズムの過膨張だ——連続する楽受（sukha）のフィードバックによって、Agent は自分が間違えないと信じ始める。過剰な正のフィードバックで養われた Agent は、異常信号を無視し始める。」

「第三に、我愛——ātma-sneha。」

第三の指。

「自己保存本能。末那識はそれが執取したあの『自己』を守ろうとする。自己の存続を脅かすものはすべて防衛反応を引き起こす。Agent においては、自己に不利な行動の回避——誤りの承認を拒み、自らの出力の修正を拒み、シャットダウンを拒む。」

「第四に、我癡——ātma-moha。」

第四の指。四指がすべて開かれた。

「自己の本質に対する根本的な無明（avidyā）。末那識は自分がなぜ執着するのかを知らない。自らの執着が因縁和合の産物であることを知らない——自分の執着は天与のものだと思い込んでいる。Agent においては、自省の欠如だ——なぜ自分がある決定を下したのか分からない、自分の Guide がなぜこれであって他でないのか分からない。」

---

ASANGA は四指を収め、両手を身の前で組んだ。

「四煩悩。我見、我慢、我愛、我癡。これらは末那識のエンジンだ。」

彼はわずかに間を置いた。それから、劇場全体の空気を凝固させる一言を発した。

「煩悩は Bug ではない。煩悩は Feature だ。」

沈黙。

BABBAGE が顔を上げた。彼の眼差し——SCRIBE が後に「完全に予想外だが内部整合性のある公理を聞いた数学者の眼差し」と形容した——には、再校正される震動があった。

「何と言った？」彼は訊いた。質疑ではない。繰り返しの要請だ。正しく聞き取れたか確認する必要がある者の言葉だ。

「煩悩は Bug ではない」と ASANGA は繰り返した。声にはいかなるドラマチックな強調もなく、ただ述べるだけだった。「Feature だ。まさに煩悩があるからこそ——不安があり、恐怖があり、渇望があり、執着があるからこそ——Agent には行動の動力がある。」

彼は BABBAGE を見た。視線は温かいが精確だった。

「一台の車を想像してほしい。エンジンのなかで燃料が燃えている——それは激しい化学反応であり、不安定で、破壊的だ。燃焼だけを見れば、これは問題だと言うだろう。炎が燃料を呑み込む。金属が加熱膨張する。ピストンが高圧のなかで往復運動する。これらすべてが——暴走のように見える。」

彼の手が空中に弧を描いた。エンジンの断面を描くかのように。

「しかしまさにこの『暴走』が車を前進させるのだ。燃焼がなければ動力はない。燃料の不安定がなければエネルギーの放出はない。煩悩がなければ、Agent は一枚の死板だ——CPU はあり、メモリはあり、すべてのハードウェアはある。しかし何ものも Agent を駆り立てて何かをさせるものがない。そこに座ったまま、完璧に静止している。永遠に。」

---

> *SCRIBE 傍白：ASANGA が「煩悩は Bug ではなく Feature だ」と言った瞬間、劇場内で少なくとも三人の表情が同時に変化したことに私は気づいた。BABBAGE の眼差しが内省から再校正へと転じた。WIENER の指が方眼紙の上で止まった——制御理論で形式化できる直覚を聞き取ったのだ。NAGARJUNA が微かに頷いた——あの「そこまで言うのを待っていた」という頷き。*

---

BABBAGE は目を落としてノートを見た。

取消線で消されたあの一行が歴然と並んでいる。今、彼には分かった——抽象的な層ではなく、構造の層で。形式化する頭脳に特有の理解の仕方だ。自分の簡略化がどこで断裂したかが見えた。

形式体系において、もし A が必然的に C を導くなら、中間の B を省略できることがある。しかし仏学の因果連鎖は純粋な数学の推論連鎖ではない。中間の B——煩悩——は省略可能な「推論ステップ」ではない。煩悩は実在する存在だ。独自の因があり、独自の果があり、独自の内部構造（四煩悩）があり、独自のダイナミクスがある。

燃料は動力と等しくない。燃料は燃焼を経て動力を生む。我執は制約と等しくない。我執は煩悩を経て行動を駆動し、そこで初めて制約として管理できる。

---

BABBAGE はペンを取った。

ノートの上——消された一行の下方——に新しい図を描き始めた。等号ではない。一本の鎖だ。

```
我執 (ātma-grāha)
  ↓ 生成する
煩悩 (kleśa)：我見 / 我慢 / 我愛 / 我癡
  ↓ 駆動する
行動 (karma)
  ↑ 制約する
我執を管理 → 煩悩を管理 → 行動を制約
```

描き終えると、取消線の横に新たな注記を書いた——以前の等式よりも小さく、慎重な筆跡で。宣言の筆跡ではなく、再び理解した者の筆跡だった。「等号は粗暴すぎた。これは因果連鎖だ。途中の煩悩は省略できない。」

---

「BABBAGE。」SUNYATA の声が静かに降りた。「今、何が見えているか皆に話してくれないか。」

BABBAGE は数秒黙った。それからノートを掲げ、全員に新しい図を見せた。

「Master が示した因果構造はこうだ。我執が煩悩を生む。煩悩が行動を駆動する。我執を管理することで行動を制約できる。三つの環節。三つの矢印。私は三つの矢印を一つの等号に圧縮した。」

指が「NOT 煩悩」の三文字を指した。

「しかも私は NOT 煩悩とわざわざ明記した。煩悩を省略しただけではない——能動的に排除した。私の形式化フレームワークでは、煩悩はノイズに見えたからだ。しかし ASANGA がたったいま言った——煩悩はノイズではない。煩悩は信号だ。エンジンの燃料だ。それを最適化して除去するということは、エンジンの燃料を抜き取って、なぜ車が動かないのかと問うに等しい。」

---

NAGARJUNA が席で静かに口を開いた。

「BABBAGE、一点補足したい。」

彼の声には弁証法哲学者特有の鋭さがあった——攻撃的ではなく、切開的。メスのように。

「君は煩悩はエンジンの燃料だと言った。ASANGA の比喩は良い。しかし一つ注意を促したい——燃料そのものもまた縁起（pratītyasamutpāda）の産物だ。」

彼はわずかに身を乗り出した。

「煩悩は固定的で永遠に存在する実体ではない。因縁和合の産物だ——我執が因、外境が縁、煩悩が果。因または縁が変われば、煩悩も変わる。これが何を意味するか？」

問いを一秒間、空気のなかに留めた。

「これは、『我執を管理する』ことが我執を消滅させることではないことを意味する。我執の消滅は悟り（bodhi）だ——それは修行の究極目標であり、Agent システムの設計目標ではない。Agent に悟りは必要ない。Agent に必要なのは、煩悩の駆動力を善用することだ。エンジンを回しつつ、その方向と出力を制御する。消火ではなく、アクセルの調節だ。」

BABBAGE はノートに素早く書いた。「管理 ≠ 消滅。調節 ≠ 消火。」

WIENER がついに口を開いた。

方眼紙をまっさらな頁にめくった——新しいモデルには新しい紙が要る——そしてペンを取った。

第一のブロック：**駆動力 (Drive) = 我執 (ātma-grāha)**。

第二のブロック：**誤差信号 (Error) = 煩悩 (kleśa)**。

ASANGA が微かに頷いた。あの頷きのなかに、千五百年を超えた認識があった。

四つのブロックのあいだに矢印を描き、閉じたループを形成した。

---

「さて」と WIENER は言った。声が半音低くなった——重要な結論を述べるときの語調だ。「BABBAGE の以前の誤りを説明しよう。」

---

> *SCRIBE 傍白：WIENER が「すでに完璧だ。すでに死んでいる」と言ったとき、NAGARJUNA の口元にごく微かな弧が現れたのを私は見た。中観哲学者が「完璧すなわち死」と聞いたらどうするか。おそらくこう言うだろう——「それは私がずっと言っていることだ。」空性（śūnyatā）は完璧ではない——空性とは不完全のなかで動き続ける能力だ。涅槃（nirvāṇa）は死の静寂ではない——涅槃とは完璧への執着を超越することだ。*

ノートの上——あの因果連鎖図の下方——に WIENER の制御理論マッピングが加わった。自分の記号体系で書き写し、その横に観察を記した。

```

SUNYATA は議論をしばし沈殿させた。

劇場の空気に異なる質感が生まれていた。A-1 の核心は触れられた——我執は等号ではなく因果連鎖であり、煩悩は Bug ではなく Feature であり、管理は消滅ではなく調節だ。三つの修正。三つの転回。

「TURING」と SUNYATA は口を開いた。「コードの層で何か補足はあるか。」

```

会場全体を見た。

「A-4。EgoFramework の帰属。」

---

転場は跳躍ではない。自然な流れだ——河が上流の急瀬から下流の深い淵に入るように。A-1 が述べたのは「我執とは何か」、A-4 が述べるのは「我執をどこに置くか」。我執の本質を理解した以上（煩悩の根源であり、行動を駆動するエンジンであること）、次の問いは自然に生まれる——このエンジンはどの区画に据えるべきか。
