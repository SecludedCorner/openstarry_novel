# 第3章：部分と全体

---

LINNAEUS はホワイトボードの前に立ち、キャップを閉じたままの青いマーカーを手にしていた。

彼は長い間、そこに立ち尽くしていた。三分か、あるいは五分か。分類学者の内的な時計において、この空白は待機ではなく、孵化であった。リンネ本人が、かつて見たことのない植物に遭遇した際、即座に命名手帳を開くのではなく、まずそこに立ち、葉の質感や萼の配列、露出した根の断面を眼差しで測ったのと同様である。まず観察せよ。完全に観察せよ。その後に命名せよ。

リンネが1735年に『自然の体系（Systema Naturae）』を出版した際、彼は界（Regnum）から種（Species）に至る階層的な分類体系を確立した。各階層には明確なセマンティック（意味論的）な境界が定められていた。界は綱を含み、綱は目は含み、目は科を含み、科は属を含み、属は種を含む。この体系の核心的な原則は、「似ているものを集める」こと——それは民俗分類学的な手法である——ではなく、「各分類ノードの意味空間は、その子ノードによって窮尽（きゅうじん）されなければならない」という点にあった。

$$	ext{属 (Genus)} = \bigcup_{i=1}^{n} 	ext{種 (Species)}_i \quad 	ext{かつ} \quad 	ext{種}_i \cap 	ext{種}_j = \emptyset \quad (i 
eq j)$$

属とは、その下のすべての種の和集合に等しい。そして、種と種は重なり合わない。これが分類学の公理である。

そして、LINNAEUS の目の前にある問題は、まさにこの公理に違反した設計であった。v0.24.0-beta の `aggregates.ts` において、`IIdentity` が識蘊（しきうん）のルート・インターフェースとして扱われていたのである。「種」の名が、「属」の位置に貼り付けられていたのだ。

彼は、この瞬間を待っていた。

---

SUNYATA が議題を発表するのを待っていたわけではない。師（マスター）の手紙が開封されるのを待っていたわけでもない。彼はもっと深いものを——Cycle 01 以降感じ続けていたが、反論するための十分な根拠を欠いていたあの不協和音が、外部の権威によって正確に言語化されるのを待っていたのである。

その不協和音は、一文に要約できた。「Identity は識蘊の全体と等しい」という一文だ。

この一文は、v0.24.0-beta の `aggregates.ts` の中で、14行のコードとしてエンコードされていた。

```typescript
/**
 * IIdentity — 識蘊 Root Interface.
 * @skandha vijnana (識蘊)
 *
 * Identity aggregate encompasses consciousness and ego framework:
 * - IGuide: Behavioral constraints and self-convergence (我執フレームワーク)
 *
 * Sanskrit: Vijñāna (विज्ञान) — consciousness.
 * Note: Guide is a behavioral constraint mechanism, not "soul".
 */
export interface IIdentity {
  /** @skandha vijnana */
  readonly skandha: 'vijnana';
}
```

クリーン。簡潔。型は正しく、テストもパスしている。

だが、それは間違っていた。

構文上の誤りではない。コンパイラが捉えるような種類の間違いでもない。それは分類学上の誤り——親ノードと子ノードの間のセマンティックな関係を理解してはじめて認識される種類の間違いであった。JSDoc には、「Identity aggregate encompasses consciousness and ego framework（同一性蘊は意識と我執フレームワークを包含する）」と記されている。しかし、「包含する」という方向が逆転していたのだ。Identity が Consciousness（意識）を包含するのではなく、Consciousness が Identity を包含する。枝の名前が幹に貼り付けられていたのである。

リンネの二名法の用語で言えば、これは属名（*Felis*）を科全体（*Felidae*）を指すために用いるようなものだ。ネコ属（*Felis*）にはイエネコとその近縁種数種しか含まれないが、ネコ科（*Felidae*）にはライオン、トラ、ヒョウ、チーター、ボブキャット、オオヤマネコなど、37の種が含まれる。*Felis* を使って *Felidae* と命名し、「どうせネコが最も代表的なのだから」と言うことは許されないのである。

