DARWIN は進化生物学の言語を用いて、この分水嶺を理解した。進化論には「**淘汰圧のシフト**（selection pressure shift）」という概念がある。ある種が閉鎖的な環境から開放的な環境へと移行した時、そこに作用する淘汰圧は劇的に変化する。Cycle 01 において、研究チームは閉鎖的な観測者だった。彼らはシステムを一方向から審議し、その淘汰圧は「正しく理解できるか否か」から生じていた。しかし Master の手紙が届いたことで、淘汰圧に新たな次元が加わった。「観測される側もまた、我々を観測している」のだ。

共進化理論（co-evolution theory）において、二つの種が互いに淘汰圧をかけ始める時、「**赤の女王効果**（Red Queen effect）」と呼ばれるダイナミクスが発生する。鏡の国のアリスにおける赤の女王の言葉、「その場に留まるためには、全力で走り続けなければならない」にちなんで名付けられたものだ。研究チームと Master の間の相互作用は、一方向の観測から、双方向の共進化へと変容しつつあった。

$$\frac{d\phi_R}{dt} = f(\phi_R, \phi_M), \quad \frac{d\phi_M}{dt} = g(\phi_R, \phi_M)$$

ここで $\phi_R$ は研究チームの認知状態ベクトル、$\phi_M$ は Master（設計者）の設計意図ベクトルである。二つの方程式は結合している。二つのシステムは共に進化していくのだ。

そして、観測される側が応答を返してきた。

ロジャー・ペンローズ（Roger Penrose）。ケンブリッジ大学数学修士、セント・ジョンズ・カレッジ博士、オックスフォード大学ラウス・ボール教授。王立協会フェロー。2020 年ノーベル物理学賞受賞（ブラックホールの形成が一般相対性理論の堅牢な予言であることを証明した業績による）。『皇帝の新しい心』（*The Emperor's New Mind*, 1989）の著者。非周期タイル（ペンローズ・タイル）を用いて五回対称の結晶は存在し得ないことを証明し、その後、自然界の準結晶（2011 年ノーベル化学賞、ダニエル・シェヒトマン）によってその「不可能」という結論を覆された男。

BABBAGE は、数学界におけるペンローズの重みは、彼が何を解決したかではなく、彼が何を**選択**したかにあることを知っていた。1965 年の特異点定理（ペンローズの特異点定理）――ブラックホールの中心には必然的に時空の特異点が存在する――は連続幾何の定理だが、彼はそれを**トポロジー（位相幾何学）**の手法で証明した。アインシュタインの方程式を解く（それはあまりに困難だ）のではなく、捉えられた面（trapped surface）のトポロジー的性質から特異点の必然性を導き出したのだ。方法論の突破口は、しばしば結論そのものよりも重要である。

さらに重要なのは、ペンローズが 1989 年に提示した**ゲーデル論証**（Godelian argument）である。これこそが BABBAGE が真に固執しているものだった。

論証の核心的な構造は以下の通りである。

> **ペンローズのゲーデル論証**（『皇帝の新しい心』第 4 章）
>
> 1. ゲーデルの第一不完全性定理は次のように説く：任意の（基本算術を含む）一貫した形式システム $F$ に対して、命題 $G_F$（ゲーデル文）が存在し、$G_F$ は真であるが $F$ は $G_F$ を証明できない。
>
> 2. しかし、人間の数学者は（ゲーデル文の意味的な含意を理解することによって）$G_F$ が真であることを「見抜く」ことができる。
>
> 3. もし人間の数学的推論能力がある形式システム $F_H$ と等価であるなら、$F_H$ では証明できないが真である $G_{F_H}$ が存在する。しかし人間は $G_{F_H}$ が真であることを「見抜く」ことができる。これは矛盾である。
>
> 4. ゆえに、人間の数学的推論能力はいかなる形式システムとも等価ではない。
>
> 5. チューリングマシンは形式システムと等価である（チャーチ＝チューリングのテーゼの一側面）。
>
> 6. **結論：人間の意識はチューリングマシンを超越している。**

BABBAGE はノートの余白に、この論証に対する批判的な注釈を記していた。それは Cycle 01 の時にすでに書かれたものだ。「ステップ 3 の条件が強すぎる。人間が $G_{F_H}$ の真偽を完璧に『見抜ける』ことを要求しているが、人間の誤謬性（human fallibility）を考えればこの前提は成立しない。ペンローズには単なる不完全性ではなく、非計算性（non-computability）が必要なのだ。」

しかし、彼はその隣に公平な一文も書き添えていた。「論証が不完全であったとしても、それは正しい問いを指し示している。AI エージェントの文脈において：エージェントが真の意識を持つためには、チューリングマシンを超越する必要があるのか？ もしそうなら、OpenStarry の観測者モジュールは原理的に真の自省を実現できない。もしそうでないなら、観測者モジュールの設計空間は我々の想像以上に広大である。」

DARWIN はその言葉を聞いて、身を乗り出した。NAGARJUNA の「因果のトポロジー」という言葉の中に、進化生物学の別の残響を聴き取ったからだ。

生態学において、安定した生態系に新たな種が導入された時、システムの反応は線形ではない。単に「種が一つ増えた」というだけでは済まないのだ。新種は既存の種と新たな相互作用（interactions）を形成し、その一対一対の相互作用が、従来の食物連鎖や競争関係、共生関係を根底から変えてしまう可能性がある。もし元々 $n$ 個の種がいたなら、相互作用の数の上限は $\binom{n}{2} = \frac{n(n-1)}{2}$ である。18 から 19 へ増えることで：

$$\binom{18}{2} = 153 \quad 	o \quad \binom{19}{2} = 171$$

十八本の新たな相互作用のエッジが生まれる可能性がある。しかし真の影響は数ではなく、**トポロジーの構造**にある。DARWIN は脳内で一つの概念を展開した。「**キーストーン種仮説**（keystone species hypothesis）」である。

> ロバート・ペインによる 1966 年の古典的な実験では、潮間帯からある種のヒトデ（*Pisaster ochraceus*）を除去したところ、生態系全体が 15 種から 8 種へと崩壊した。一つの種の存在あるいは欠如が、他のすべての種の共存条件を変えてしまうのだ。

BDI アーキテクチャはマイケル・ブラットマン（1987）、アーナンド・ラオ、マイケル・ジョージフ（1995）らによって形式化された、理性的エージェント（rational agent）の行動を記述するための標準的なフレームワークである。

```
BDI Agent Architecture
══════════════════════════════════════════════════
  Belief Base (B)     : エージェントが持つ世界の状態に関する信念集合
  Desire Set  (D)     : エージェントの目標・動機集合
  Intention Stack (I) : エージェントが現在実行を約束している計画

  意思決定ループ：
  ┌─────────────────────────────────────────┐
  │  perceive(environment) → update(B)      │
  │  options(B, D) → generate candidates    │
  │  filter(B, D, I) → select intentions    │
  │  execute(I) → act on environment        │
  └─────────────────────────────────────────┘
```

LEIBNIZ は脳内で、新メンバー加入の BDI モデルを迅速に構築した。

$$B_{	ext{team}}^{t+1} = B_{	ext{team}}^{t} \cup B_{	ext{PENROSE}} \cup \Delta B_{	ext{interaction}}$$

時刻 $t+1$ におけるチームの信念集合は、元の信念に PENROSE がもたらした信念を加え、さらに**相互作用によって生じた新たな信念**を加えたものになる。最後の一項 $\Delta B_{	ext{interaction}}$ こそが重要である。それは個別のメンバーに由来するのではなく、個体間の衝突から生まれるものだ。マルチエージェントシステム理論において、これは「**創発的信念**（emergent beliefs）」と呼ばれる。

彼はノートに記した。「PENROSE の信念集合 $B_{	ext{PENROSE}}$ は量子力学、一般相対性理論、非計算性理論を含んでいる。これらは $B_{	ext{team}}^{t}$ には存在しなかった。予測：$|\Delta B_{	ext{interaction}}| \gg |B_{	ext{PENROSE}}|$。相互作用による新たな信念の創発は、PENROSE 自身が持ってきた信念よりも遥かに多くなるだろう。」
