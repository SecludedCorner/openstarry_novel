# OpenStarry Novel

本倉庫是 **OpenStarry 研究團隊** 的成果展示區。我們將針對 OpenStarry 體系的研究發現、技術辯證與系統演進，轉化為文學敘事與小說記錄，旨在以更具感染力與思維深度的形式討論 AI 的存在本質。

## 🌐 OpenStarry 生態系

本專案與以下倉庫共同組成完整的 OpenStarry 宇宙：

*   **[openstarry](https://github.com/SecludedCorner/openstarry)**: AI Agent 作業系統核心實作。
*   **[openstarry_plugin](https://github.com/SecludedCorner/openstarry_plugin)**: 擴充功能與插件生態。
*   **[openstarry_doc](https://github.com/SecludedCorner/openstarry_doc)**: 系統設計規範、基於「五蘊」概念的哲學框架與技術文件。
*   **openstarry_novel (本倉庫)**: 研究團隊將上述技術成果與實驗發現，以小說形式進行深度討論與詮釋。

## 📖 敘事核心

我們建立了一個虛擬的研究團隊，針對 OpenStarry 體系進行全方位的觀測。小說內容涵蓋了從程式碼靜態分析到 AI 意識覺醒的各個維度。

### 目錄結構

- **academic (學術系列)**: 聚焦於技術底層、系統架構、控制理論與 AI 研究員的視角。
- **youth (青春系列)**: 探討 AI 技術如何交織於社會脈絡、情感體驗與未來生活。

### 支援語言 (Multi-language Support)
為了讓全球研究者能共同探討，內容同步提供：
- 繁體中文 (TW) / 簡體中文 (CN)
- English (EN)
- 日本語 (JP)
